Создание интерактивных графиков с использованием Plotly.js, часть 4: диаграммы пузырьков и точек...

Code Graphs & Charts
Tutorial

До сих пор в серии вы узнали, как создавать линейные диаграммы и гистограммы в Plotly.js. Как я упоминал во вводном руководстве этой серии, Plotly.js не ограничивается только несколькими типами диаграмм. Существует более 20 различных типов диаграмм, которые вы можете создать с помощью этой библиотеки. Каждый из этих типов диаграмм имеет свой собственный набор параметров настройки, а также атрибуты, которые являются общими для нескольких диаграмм.

В этом уроке вы узнаете, как создавать графики пузырьков и точек в Plotly.js. Оба этих типа диаграммы используют те же трассы scatter, которые мы использовали при создании линейных диаграмм. Важным отличием на этот раз является то, что атрибут mode будет установлен на markers.

Создание пузырьковых диаграмм в Plotly.js

Диаграммы пузырьков используются для отображения трех измерений данных на диаграмме. Данные, связанные с каждым объектом, строятся с использованием пузырьков или дисков, где положение диска по оси x-y указывает его значения x-y, а площадь диска используется для построения значения третьей точки данных. Диаграммы пузырей - это вариация диаграмм рассеяния. По этой причине имеет смысл установить атрибут type в scatter при создании пузырьковой диаграммы.

Есть несколько вещей, которые вы должны помнить при создании пузырьковых диаграмм. Во-первых, третье значение данной точки данных представлено площадью пузырька, а не его радиусом. Радиус фактически пропорционален квадратному корню фактического значения. Во-вторых, вы можете использовать их только для построения положительных значений. Это имеет смысл, потому что область пузыря не может быть отрицательной или нулевой.

Теперь давайте создадим нашу первую диаграмму пузырьков, установив атрибут mode в markers. Все атрибуты, управляющие появлением пузырьков в диаграмме, присутствуют в объекте-маркере. Атрибут size объекта marker может использоваться для указания размера пузырьков в качестве третьей точки данных. Этот атрибут может быть либо числом, либо массивом чисел. При создании пузырьковых диаграмм редко бывает, чтобы все объекты имели одинаковое значение для своей третьей точки данных. Таким образом, вы обычно устанавливаете атрибут size как массив.

Прозрачность всех пузырьков можно контролировать, используя атрибут opacity. Подобно size, этот атрибут также принимает значения как число или массив или числа со значениями от 0 до 1. В случае линейных диаграмм значение непрозрачности по умолчанию для разных пузырьков или дисков было равным 1. В случае пузырьковых диаграмм значение непрозрачности по умолчанию становится равным 0,7.

Другим очень полезным атрибутом, который можно использовать для точного создания пузырьковых диаграмм, является параметр sizemode. Этот атрибут определяет, следует ли считать значение, указанное в атрибуте size, областью пузыря или его диаметром. Значение по умолчанию для sizemode - это diameter. Однако, если вы создаете пузырьковые диаграммы, имеет смысл установить значение sizemode в area. Таким образом, вам не придется выполнять какие-либо вычисления, чтобы определить правильный размер пузыря самостоятельно. Вы можете использовать следующий код для создания пузырьковой диаграммы с использованием параметров, которые мы только что обсуждали.

var bubbleDiv = document.getElementById("bubble-chart");var traceA = {  type: "scatter",  mode: "markers",  x: [5, 13, 24, 35, 46, 60],  y: [40, 70, 65, 15, 75, 49],  marker: {    size: [100, 200, 800, 600, 500, 600],    sizemode: "area"  }};var data = [traceA];var layout = {  title: "A Bubble Chart in Plotly"};Plotly.plot(bubbleDiv, data, layout);

Прямо сейчас, каждый пузырь имеет тот же цвет. Цвет отдельных пузырьков или всей трассы можно изменить сразу, используя атрибут color.

Другой способ установить цвет различных пузырьков - использовать цветные полосы. В таких случаях цвет пузырька определяется на основе числового значения, указанного в массиве, которое вы передали атрибуту color объекта marker. Эта полоса цвета может выступать в качестве четвертой точки данных при построении графика.

Например, предположим, что вы замышляете диаграмму пузырьков, которая показывает местоположение и область различных парков в большом городе. В этой ситуации идеально подходит пузырьковая диаграмма. Однако плотность деревьев каждого парка или среднее число людей, посещающих каждый парк, могут быть разными. Если цвет каждого пузырька указан как числовое значение, можно будет присвоить отдельный цвет отдельным пузырькам для построения значения четвертой точки данных.

В нашем примере цвет, соответствующий разным значениям плотности деревьев, может быть задан с использованием атрибута colorscale. При использовании этого атрибута вам необходимо как минимум указать сопоставление для наименьших (0), а также самых высоких (1) значений. Примером допустимого значения scalecale является [[0, "rgb (0, 0, 0)"], [1, "rgb (0, 255, 0)"]]. У вас также есть возможность использовать строку имени палитры в качестве значения colorscale. Grays, YlGnBu, Greens, YlOrRd, Bluered, RdBu, Reds, Blues, Picnic, Rainbow, Portland, Jet, Hot, Blackbody, Earth, Electric и Viridis считаются действительными.

Вы можете указать наименьшее числовое значение цвета, к которому должно быть сопоставлено самое низкое значение цветовой шкалы с использованием атрибута cmin. Аналогичным образом вы также можете указать самое высокое значение цвета, значение которого должно быть сопоставлено с самым высоким значением цветовой шкалы с использованием атрибута cmax. Можно изменить отображение цвета, установив значение параметра reverseescale в значение true. В этом случае самый низкий цвет будет сопоставлен с самым высоким значением, а самый высокий цвет будет сопоставлен с самым низким значением.

Объект colorbar предоставляет множество параметров, которые можно использовать для управления его внешним видом. Ширину цветовой полосы, созданной Plotly.js, можно контролировать с использованием параметра thickness. Толщина указана в пикселях, а значение по умолчанию - 30. Длина цветовой полосы, за исключением прокладки на обоих концах, может быть указана с использованием атрибута len. Положение x и y в цветовой полосе можно задать с помощью параметров x и y. Позиция определяется в терминах фракций участка, и ее действительное значение может находиться где-то между -2 и 3. Значение по умолчанию для x равно 1,02, а значение по умолчанию для y равно 0,5. Справа x и y цветовой полосы можно управлять аналогичным образом с помощью параметров xpad и ypad. Вы можете присвоить заголовок цветовой полосе, используя атрибут title. Позиция этого названия может быть указана в отношении цветовой полосы, используя параметр titleside. Допустимые значения для этого параметра: top, bottom и right. Значение по умолчанию - top.

Следующий код создаст пузырьковую диаграмму, в которой темнота зеленого цвета отображается на плотность дерева в этом парке.

var bubbleDiv = document.getElementById("bubble-chart");var traceA = {  type: "scatter",  mode: "markers",  x: [5, 13, 24, 35, 46, 60],  y: [40, 70, 65, 15, 75, 49],  marker: {    color: [35, 10, 50, 40, 18, 30],    colorscale: [[0, "rgb(200, 255, 200)"], [1, "rgb(0, 100, 0)"]],    cmin: 0,    cmax: 50,    size: [600, 1200, 800, 400, 1500, 2000],    sizemode: "area",    showscale: true,    colorbar: {      thickness: 10,      y: 0.5,      ypad: 0,      title: "Tree Density",      titleside: "bottom",      outlinewidth: 1,      outlinecolor: "black",      tickfont: {        family: "Lato",        size: 14,        color: "green"      }    }  }};var data = [traceA];var layout = {  title: "A Bubble Chart in Plotly"};Plotly.plot(bubbleDiv, data, layout);

Создание точки в Plotly.js

Точечный график - это график, который используется для построения точек в очень простом масштабе. Вы должны иметь в виду, что точечные графики подходят только для небольшого количества данных. Построение большого количества точек сделает диаграмму очень загроможденной. Точно так же, как пузырьковые диаграммы, точечный график также требует, чтобы атрибут type в значении scatter, а атрибут mode - markers. Так как в точечных графиках также используются маркеры для построения точки, все параметры настройки для них также доступны под объектом marker.

Следующий код создает точку, которая показывает оценки, полученные двумя лучшими учениками на экзамене каждый год. Я также использовал некоторые другие атрибуты, связанные с легендой и осью х, чтобы изменить общий внешний вид нашего графика.

var dotDiv = document.getElementById("dot-chart");var traceA = {  type: "scatter",  mode: "markers",  x: [2011, 2012, 2013, 2014, 2015, 2016],  y: [789, 795, 760, 775, 780, 783],  name: "Highest Marks",  marker: {    color: "rgba(156, 165, 196, 0.5)",    line: {      color: "rgba(156, 165, 196, 1)",      width: 1,    },    symbol: "circle",    size: 20  },  hoverlabel: {    bgcolor: "black",  }};var traceB = {  type: "scatter",  mode: "markers",  x: [2011, 2012, 2013, 2014, 2015, 2016],  y: [769, 755, 747, 770, 771, 781],  name: "Second Highest Marks",  marker: {    color: "rgba(165, 196, 50, 0.5)",    line: {      color: "rgba(165, 196, 50, 1)",      width: 1,    },    symbol: "circle",    size: 20  },  hoverlabel: {    bgcolor: "black",  }};var data = [traceA, traceB];var layout = {  title: "Marks Obtained by Top Two Students",  xaxis: {    showgrid: false,    showline: true,    linecolor: "rgb(200, 0, 0)",    ticks: "outside",    tickcolor: "rgb(200, 0, 0)",    tickwidth: 4  },  legend: {    bgcolor: "white",    borderwidth: 1,    bordercolor: "black",    orientation: "h",    xanchor: "center",    x: 0.5,    font: {      size: 12,    }  },  paper_bgcolor: "rgb(255, 230, 255)",  plot_bgcolor: "rgb(255, 230, 255)"};Plotly.plot(dotDiv, data, layout);

Мы уже рассмотрели различные атрибуты объекта marker в предыдущем разделе. На этот раз я внес некоторые изменения в схему диаграммы. Использование атрибута title для присвоения заголовка вашему графику сделает его более информативным, быстро информируя зрителей о том, что представляет собой диаграмма. Я также изменил внешний вид линии оси x, чтобы он выделялся из других линий. Как вы можете видеть, ширину и цвет тиков можно контролировать с помощью атрибутов tickcolor и tickwidth.

Положение легенды можно указать с помощью атрибутов x и y. Оба эти числа принимают значение от -2 до 3. Позиция определяется в терминах фракций. Якорную точку для позиций x и y можно указать с помощью атрибутов xanchor и yanchor. Эти два атрибута определяют точки, которые следует использовать в качестве ссылки для измерения расстояния, заданного атрибутами x и y. В приведенном выше примере расстояние 0.5 от атрибута x измеряется от центра легенды, потому что xanchor был установлен в center.

Заключение

В этом учебнике показано, как создавать диаграммы пузырьков и точек в Plotly.js, используя атрибуты одного и того же объекта маркера. Вы также узнали, как добавить дополнительную информацию в свои пузырьковые диаграммы с помощью цветовых полос и установить цветовую шкалу этих баров в соответствии с вашими собственными требованиями.

Я попытался охватить почти все атрибуты маркера, которые вам, возможно, придется использовать в ваших проектах. Если вам нужно больше узнать о создании диаграмм с пузырьками и точками, вы можете перейти в раздел ссылок маркера на сайте Plotly.js.
