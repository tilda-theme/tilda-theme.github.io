Выучить что-то новое: разбираемся с этими популярными инструментами к 2018 году ...

Web Design CSS
Tutorial

Новый год — отличное время, чтобы выучить что-то новое. Если Вы хотите оказаться на шаг впереди других — ознакомьтесь с нашим топом самых свежих инструментов для 2018 года.

1. CSS Сетка

"Что можно назвать самой важной вещью в веб-дизайне, происходящей в данный момент? Это. Мощная сетка появилась в веб среде."—Chris Coyier, сооснователь CodePen

Подход к размещению элементов на странице меняется, и на передовой этих изменений находится CSS Сетка. Это краткое введение, плюс эти курсы помогут Вам быстро со всем разобраться, прямо сейчас.

Новая Css разметка

Хотите иметь преимущество в этот важнейший момент в развитии адаптивного дизайна? Новая книга Рэйчел Эндрю совместно с "A book Apart", "Новая CSS разметка",— отличное вводное пособие в сложную тему. На практических примерах Рэйчел дотошно разбирает все детали спецификации в доступной форме.

2. Vue.js

Vue.js набирает обороты и уже зарекомендовал себя, как достойного конкурента React"у. Согласно ежегодному опросу по использованию JavaScript (annual State of JavaScript survey), Vue и React имеют самые положительные рейтинги от разработчиков среди любых других JavaScript библиотек или фреймворков.

Одним из главных преимуществ Vue является то, что им легко овладеть, зная только HTML и CSS. Разносторонность Vue позволяет одинаково комфортно работать как с маленькими, так и с большими проектами. В дополнение ко всему, он невероятно быстр!

Следующие курсы очень быстро научат Вас кодить на Vue!

    Lawrence Turton
  • Vue.js

    Derek Jensen

3. Adobe XD

Adobe XD быстро становится графическим инструментом номер один в среде веб-дизайнеров. Он подходит для создания дизайна, прототипирования или же совместной работы с другими людьми, и все из одного приложения!

Освоиться с Adode XD и научиться созданию дизайна иконок, дизайна Landing Page, прототипированию, как и многому другому возможно с помощью этих четырех курсов.

4. Основы JavaScript

Экосистема JavaScript сейчас стала невероятно объемной, и настал неизбежный момент для самого языка стать лучше. Согласно ежегодному опросу по использованию JavaScript (annual State of JavaScript survey), большая часть разработчиков считает, что JavaScript, несмотря на все свои недостатки, движется в правильном направлении.

Не лучшее ли это время, чтобы разобраться с самыми основами?

  • JavaScript
    Современные основы JavaScript
    Dan Wellman

Вам также следует ознакомиться с нашим исчерпывающим руководством по разработке с помощью JavaScript, являетесь ли Вы начинающим веб-разработчиком, либо же хотите изучить более сложные аспекты, такие как использование новейших фреймворков, или даже построение своих!

5. Доступность для каждого

Настало время, когда веб должен стать местом, доступным каждому. Книга Лоры Кальбег совместно с "A Book Apart", "Доступность для каждого", несет в себе идею того, что сделать интернет местом, где будет удобно каждому, задача любого веб-профессионала.

Она проворно проводит вас через пространство общедоступности таким образом, что Вам становятся понятны проблемы, с которыми сталкиваются люди с ограниченными возможностями; вы получаете набор важных правил и инструкций; и учитесь тому, как создать доступный каждому дизайн.

Начальное руководство по всеобщей доступности веба

Вы можете разобраться с доступностью веба для каждого и сразу же преуспеть в этом деле с помощью нашего курса, Начальное руководство по Веб-доступности.

Являетесь ли Вы дизайнером, разработчиком, начальником проекта или исполнительным директором, этот курс поможет Вам осознать, что доступность интернета для каждого стоит потраченных на себя усилий, к тому же делать приходится не так уж и много, если заняться доступностью на ранних этапах проекта.

  • Доступность веба
    Начальное руководство по доступности веба
    John Hartley

6. Создание дизайна в браузере

Веб-приложения, позволяющие создавать дизайн прямо в браузере, становятся все более и более популярными, как и их подлинная ценность для веб-разработчика. Конечно, они не обладают полным набором функционала десктоп-приложений, однако несмотря на это, они все еще могут быть достаточно полезными, к тому же они часто имеют функционал, недоступный обычным десктоп-приложениям.

Прочитайте наше руководство по созданию веб-дизайна в веб-приложениях, либо же научитесь работать с Gravit Desiner или Figma в наших курсах.
