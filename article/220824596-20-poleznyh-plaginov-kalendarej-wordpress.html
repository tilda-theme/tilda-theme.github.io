20 полезных плагинов календарей WordPress...

Web Design WordPress
Tutorial

Первое, что мне приходит в голову, когда я слышу «WordPress» и «calendar» - это виджет календаря по умолчанию, с сообщением WordPress по дате.

Но WordPress calendar plugins охватывают гораздо больший - и более мощный - массив плагинов. Это calendar плагины для событий, заказов и встреч. Некоторые превращают WordPress в собственное рабочее приложение, в то время как другие предлагают нужный набор инструментов для создания полностью функционального расписания и платежного шлюза.

Взгляните на эти 20 плагинов календаря WordPress, доступных на рынке Envato, и вы поймете, что я имею в виду.

1. EventOn - WordPress Event Calendar Plugin

Плагин календаря событий EventOn WordPress - отличное решение под ключ, которое не скупится на возможности.

Он имеет чистый, минимальный дизайн для быстрой и лёгкой работы.

Полностью оптимизированный для мобильного использования, этот крошечный календарь отличается большими возможностями:

  • Использовать улучшенные изображения
  • Добавить свои поля данных
  • Совместить с Google Maps
  • Генерировать shortcode и calendar виджеты
  • Назначать цвета по категориям событий
  • и многое другое

Вы можете не только настраивать каждое событие с помощью изображений, карт, значков, цветов, но и настроить календарь для поиска и сортировки несколькими способами.

EventOn - WordPress Event Calendar Plugin - действительно хороший, простой в использовании плагин календаря событий с уникальным slider addon.

2. Pinpoint Booking System PRO - Book Everything with WordPress

Это больше, чем просто календарь или календарь событий.

Система Pinpoint Booking System - это полная система бронирования, которая позволяет легко интегрироваться с WooCommerce.

Создание системы бронирования и заказов никогда не было таким простым.

Особенности:

  • расширенный поиск
  • бронирование даты и/или часа
  • поддержка нескольких языков и валют
  • простые формы подачи заявок на бронирование
  • скидки, купоны, ваучеры, налоги и сборы
  • и многое другое

Отвечает всем требованиям, а также позволяет настраивать шаблоны электронной почты и уведомления.

Pinpoint Booking System PRO - Book everything with WordPress - всё, что вам нужно для создания впечатляющей системы бронирования, не выходя из панели администратора WordPress.

3. Bookly - Book Appointments, Book Services, Book Anything! Быстрое и простое бронирование для ваших клиентов!

Вы можете заказать и запланировать что угодно с этим плагином WordPress.

Bookly - невероятная система бронирования встреч, которая стирает границы между веб-сайтом WordPress и веб-приложением.

Хороший дизайн, отзывчивость и поддержка жестов.

Вы также найдете:

  • конструктор форм
  • синхронизацию с Google Calendar
  • интеграцию с WooCommerce
  • уведомления по SMS и электронной почте
  • несколько вариантов онлайн-платежей
  • и много, чего ещё

Клиенты могут планировать встречи, включать личную информацию и отправлять платежи. Они могут отменить свои встречи через электронную почту с подтверждением.

Есть возможность создавать расписание для каждого сотрудника - вы даже можете блокировать праздники.

Возможности настройки Bookly и полезные функции сделают его более, чем удобным способом обслуживания клиентов и помогут владельцу сайта отслеживать персонал и встречи.

4. Calendarize it! для WordPress

Несмотря на то, что этот плагин календаря WordPress построен для работы в связке с Visual Composer, он легко работает и сам по себе.

Calendarize it! Для WordPress полно функций, которые можно настроить так, как вам нравится.

С очень простым дизайном, этот плагин календаря может вписаться в любую схему или стиль.

Вы найдете много полезных функций, включая, но не ограничиваясь:

  • события с участием пользователя
  • sidebar виджеты и shortcodes
  • поддержка настройки Post Types
  • расширенная фильтрация по схеме
  • отдельные события, повторные события, произвольные события и повторные события с исключениями

Вы можете настроить детали места встречи, интеграцию с Google Maps и использовать на международном уровне с несколькими языками и форматами дат.

Calendarize it! for WordPress - один из самых гибких настраиваемых плагинов календаря WordPress.

5. WordPress Pro Event Calendar

Есть функции, которых вы ожидаете от плагина календаря WordPress; однако, есть те, которые действительно выделяются.

WordPress Pro Event Calendar - это плагин календаря WordPress, который стоит особняком.

Хорошо спроектированный и полностью отзывчивый, обладает действительно замечательными функциями:

  • WPML поддержка
  • гибкая настройка событий
  • интеграция с Google Maps
  • подписка на календарь
  • настройка полей и поддержка диапазона дат
  • и прочее

Но что его отличает, это возможность импортировать события из каналов ICS и Facebook.

Лучше всего, что WordPress Pro Event Calendar принимает события от внешних пользователей, что позволяет легко планировать свои.

6. Event Booking Pro - WP Plugin [PayPal or Offline]

Вот еще один отличный вариант для создания полностью функциональной системы бронирования событий.

Event Booking Pro очень pro, поскольку он может похвастаться более, чем 500 настройками!

У этого плагина календаря много функций WordPress и они всё добавляются. Вот некоторые:

  • безлимитные события
  • контрольная панель AJAX
  • автономный и многократный заказ
  • PayPal, coupon, и CSV интеграция
  • настройка email отправлений бухгалтеру и админу
  • и ещё очень и очень много

Там нет недостатка в шорткодах, он поддерживает CSV, и всё можно настроить, как вам нравится, что вписывает его в дизайн темы WordPress отлично.

Event Booking Pro - WP Plugin [PayPal or Offline] впечатляющий плагин календаря WordPress.

7. Timetable Responsive Schedule For WordPress

Timetable Responsive Schedule For WordPress предлагает совершенно иной взгляд и подход к плагинам календаря WordPress.

Легко создавайте расписание событий!

Используйте генератор короткого кода Timetable для создания расписания занятий, медицинских отделов, ночных клубов, обучения и т. д.

Быстрая настройка событий с:

  • добавление нового события с указанием дня, времени, категории и т. д.
  • настройка дизайна и внешнего вида
  • создание шорткода для сообщения или на страницу

Вот и всё!

Существует множество опций событий и множество вариантов shortcode, включая выбор цвета и шрифта.

Timetable Responsive Schedule For WordPress имеет отличную линейку виджетов, выводящих его в лучшие плагины календаря WordPress, которые вы найдете.

8. Booked - Appointment Booking for WordPress

Booked - Appointment Booking for WordPress ещё один надежный вариант бронирования.

Если вы ищете место для бронирования, подумайте о бронировании Booked.

Мобильный, с чистым дизайном, этот плагин имеет некоторые приятные функции:

  • бронирование
  • настройки страниц профиля
  • настраиваемые поля и временные интервалы
  • настройка emails пользователя и админа
  • отображение календаря в виде shortcode или виджета
  • и более

Мне очень нравятся страницы профиля клиента, но возможность отображать несколько агентов бронирования с назначенными датами - это то, чем этот плагин календаря WordPress действительно светит.

Если возьмёте Booked - Appointment Booking for WordPress, смотрите и приложения Booked: Front-End Agents, Payments with WooCommerce, и Calendar Feeds.

9. WordPress Events Calendar Registration & Booking

The WordPress Events Calendar Registration & Booking плагин обладает богатой функциональностью, предлагая такие опции, которые бывают только в дополнениях.

Здесь всё, что нужно (возможно, больше) для успешной настройки полнофункциональной системы регистрации и бронирования.

Это решение включает в себя «из коробки»:

  • shortcodes
  • палитру цветов
  • языковую поддержку
  • интеграцию с Google Maps
  • повторяющиеся события, билеты и купоны
  • и прочее

Одна из лучших функций этого плагина и других - таймер обратного отсчёта для всех ваших событий.

Красиво оформленный с привлечением Bootstrap 3.0, WordPress Events Calendar Registration & Booking нельзя пропустить.

10. Booki - Booking Calendar Plugin - A WordPress Booking System for Online Reservation and Appointments

Как и некоторые из упомянутых плагинов календаря WordPress, Booki предлагает полную систему бронирования с использованием WordPress.

Существует много замечательных приложений для этих типов плагинов, каждый из которых предлагает уникальную перспективу процессом управления.

Неограниченное бронирование и поставщики услуг. Имеются опции:

  • дополнительные опции для заказов
  • online платежи через PayPal Express
  • offline платежи — бронь сейчас, оплата после
  • страница статистики администратора со сводной информацией
  • синхронизация заказа с Google Calendar поставщика
  • и другое

Что касается самого календаря, он отображает вашу бронь и назначения во всплывающем окне, развёрнуто и т.д.

Booki это отличное решение для бронирования нескольких проектов и поставщиков услуг.

Team Booking - WordPress Booking System это новый подход к бронированию и использованию Google Calendar в полной мере.

12. WP Booking Calendar

Многие плагины календаря WordPress включают богатый набор функций, предлагающий надёжное решение «всё-в-одном».

WP Booking Calendar имеет необходимый, освежающе простой подход.

В минимальной конфигурации вы можете настроить доступные временные интервалы, неограниченные календари, подтверждения и многое другое.

Опции включают:

  • и другое

Вы можете указать количество дней до того, как пользователь сможет сделать заказ.

WP Booking Calendar прост без потери больших возможностей.

13. WordPress Multiple Events Calendar with Countdown

Немногие плагины календаря WordPress предлагают обратный отсчёт времени до событий.

Но WordPress Multiple Events Calendar with Countdown это делает!

Это очень простой плагин календаря:

  • готовый виджет
  • предпросмотр вживую
  • тёмная и светлая оболочки
  • более пяти событий ежедневно
  • установка точного времени события (часы и минуты)

WordPress Multiple Events Calendar with Countdown добавляет время ожидания по таймеру (расположен под календарем) до следующего события.

14. Timetable for WordPress

Календарь расписания - отличный вариант для ситуаций, когда в один и тот же день происходит несколько событий.

Расписание для WordPress имеет приятный подход в стиле TV руководства в презентации контента.

Этот плагин на jQuery и CSS3, позволяет добавлять события и изменять расписание его современному администратору.

Одна из мощных функций Timetable for WordPress возможность импорта и экспорта через CSV. Вы можете редактировать и вносить изменения в Excel или другом приложении и импортировать ваши изменения и добавленные события.

Можете выбрать программу для печати пользователям, которые предпочитают расписание на бумаге.

Этот плагин полностью редактируем и может использоваться двумя способами: базовым и расширенным.

  • и прочее

Chronosly Event Calendar WordPress Plugin удобно настроить в вашей панели админа WordPress.

Но Business Calendar - WordPress Internal Calendar создан как полнофункциональный внутренний календарь для каждого пользователя.

20. Facebook Events Calendar WordPress Plugin

Приятно закончить длинный список плагинов WordPress calendar чем-то уникальным.

Facebook Events Calendar WordPress Plugin соединит календарь сайта с вашим Facebook Events.

Отображение событий с вашей страницы Facebook, профиля и/или групп. С официальным Facebook Graph API, у вас будет:

  • caching на стороне сервера
  • поддержка виджетов
  • настройка CSS (предоставлен файл LESS)
  • много языков с построчным переводом
  • поддержка нескольких страниц и профилей в календаре
  • и прочее

Он имеет отличную производительность с caching на стороне сервера, а также настройки конфигурации. Вы можете выбрать использование даты в календаре или часового пояса браузера.

Facebook Events Calendar WordPress Plugin уникальный путь переноса Facebook Events на свой сайт.
