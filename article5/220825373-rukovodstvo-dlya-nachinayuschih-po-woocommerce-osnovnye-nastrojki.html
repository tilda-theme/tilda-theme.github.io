В предыдущей статье мы рассмотрели введение в WooCommerce. Теперь давайте поговорим об основных настройках WooCommerce.

Установка и активация

Это основанный на WordPress плагин; его установка очень проста и может быть выполнена в несколько кликов. Для тех новичков, кто не знает, как установить плагин WordPress, вот как это сделать:

  • Зайдите в панель управления WordPress.
  • Кликните меню Плагины и выберите Добавить новый.
  • Введите в поиске WooCommerce.
  • Вы увидите его первым в списке результатов.
  • Нажмите Установить.
  • После установки нажмите Активировать плагин.

Инициализация

Сразу после установки в верхней части окна будет отображаться сообщение (пурпурного цвета), которое приветствует пользователя и говорит, что «Вы готовы начать продажи :)»; в этот момент WooCommerce предлагает два варианта: вы можете «Установить страницы WooCommerce» или «Пропустить установку».

Нажав кнопку Установить страницы WooCommerce, вы можете установить несколько страниц (страница входа, оплаты покупки и т.д.), которые обеспечивают необходимую функциональность этого плагина.

Начало работы с WooCommerce

Вы увидите два новых меню в панели управления администратора:

  • WooCommerce
  • Товары

Давайте начнём наше обсуждение с меню WooCommerce, из которого мы перейдём в подменю Настройки.

Настройки

Здесь вы можете настроить свой интернет-магазин. В этом разделе плагина вы можете найти различные типы настроек: общие, товары, налог, оплата, доставка, аккаунты и электронная почта. Давайте сначала рассмотрим Основные настройки.

Основные настройки

В раздел основных настроек включены несколько простых параметров. Давайте настроим каждый из них.

Основные настройки

На момент написания этой статьи здесь находится четыре общих параметра:

1. Базовое местоположение

2. Торговая точка

WooCommerce позволяет вам выбрать, с какими странами вы готовы вести торговлю. Здесь вы можете выбрать вариант продажи товаров во все страны или только в определённые. Всё, что вам нужно сделать — это ввести название в поле Страны для их выбора. Этот список конкретных стран появится, только если вы выберете вариант Продавать только в определённых странах.

3. Предупреждение магазина

4. API

С момента выпуска WooCommerce v2.1 одно из лучших изменений, которые были сделаны, является введение REST API. Включение этого флажка позволит разработчикам взаимодействовать с различными вызовами API, которые представляют данные вашего веб-магазина.

Этот параметр только для чтения может предоставить доступ к вашим данным в формате XML или JSON. Можно получить доступ к пяти основным ресурсам через этот интерфейс API, т.е. заказ, клиент, купон, товар и данные отчётов.

Настройки валюты

  1. Валюта. WooCommerce предоставляет огромный диапазон настраиваемых валют, который позволяет вам выбрать символ или метку существующей валюты. Как обсуждалось ранее, настройка базового местоположения определяет налоговые ставки по умолчанию, поэтому настройка вашей родной валюты сделает настройки ещё более детальными. Это помогает контролировать, в какой валюте цены будут указаны в каталоге и в какой валюте шлюзы будут принимать платежи.
  2. Позиция валюты. Этот плагин предлагает много вариантов для настройки валюты, и позиция валюты является одним из них. Вы можете выбрать такое положение валюты по умолчанию для ваших цен, как Слева, Справа, Слева с пробелом и Справа с пробелом.
  3. Тысячный разделитель. Этот параметр позволяет пользователю задать разделитель тысяч для отображаемых цен. Например можно задать (,) как тысячный разделитель, который будет отображаться в ценах как 1,000.
  4. Десятичный разделитель. Можно задать разделитель для цен, отображаемых с этим знаком. Например, можно задать (.) как десятичный разделитель, который будет отображаться в ценах как 10.00.
  5. Количество знаков десятичной дроби. Этот параметр используется для того, чтобы украсить отображаемые цены. Установка значения (2) позволит отображать только два десятичных знака в ценах, например 999,99.

Стили и скрипты

1. Стили интерфейса

Прокрутите до настроек WooCommerce Стили интерфейса прямо под разделом Стили и скрипты. Они позволяют вам по собственному выбору задать цвета, подходящие к нуждам вашего интернет-магазина. WooCommerce предлагает пять различных стилей интерфейса: Основной, Дополнительный, Выделение, Содержимое и Подтекст. Каждый из этих стилей имеет собственное значение.

  • Основной используется для оформления кнопок призыва к действию (CTA), фильтра цен, и т.д.
  • Дополнительный используется для различных кнопок и вкладок, которые могут быть стилизованы с этим параметром.
  • Выделение используется для различных меток цен и объявлений о распродаже.
  • Содержимое: с этим параметром может быть стилизован фон страницы.
  • Подтекст позволяет добавить дополнительный текст, такой как навигация, примечания и т.д., а затем эта опция позволяет вам выбрать цвет для его оформления.

2. Скрипты

Прямо в конце вы можете найти две особых настройки для пункта Скрипты. Вот они:

  • Включить Lightbox. Если я думаю, как покупатель, то я определённо хочу иметь полный вид товара, который я планирую купить. Имея это в виду, хороший продавец всегда загружает галерею изображений. С флажком Включить Lightbox изображения продуктов будут открываться во всплывающих окнах, когда пользователь кликает по ним.
  • Включить расширеный список выбора страны. Этот параметр поможет любому клиенту при оформлениии заказа. Включение этого скрипта позволит пользователю производить поиск в поле выбора страны.

Как только вы закончите с изучением всех этих настроек, вы можете нажать на кнопку «Сохранить изменения» внизу. На этом завершим конфигурацию общих настроек в WooCommerce.
