20 полезных iOS иконок и ресурсов с шаблонами приложений...

Code Mobile Development
Tutorial

Как разработчик, ваши дизайнерские навыки могут быть не такими уж великолепными. Но вы хотите, чтобы приложения выглядели великолепно. Правда? В этом кратком посте я перечислю 20 графических ресурса для iOS, которые вы сможете найти на GraphicRiver. Они не только сэкономят вам время, но и сделают ваши приложения великолепными.

1. Иконки приложений

Иконки приложений один из самых важных частей приложения. Следует потратить некоторое время на поиск правильного внешнего вида. Следующие пункты смогут помочь вам с этим.

App Icon Creator With Flat Shadow Generator

Этот включает более, чем просто иконки. Он включает несколько действий для Photoshop для создания трёх стилей иконки:

  • стандартный стиль iOS 7
  • шикарный плоский стиль
  • стиль плоского тени

В дальнейшем иконки можно изменять с использованием градиентов, цвета и сетки, которая помогает позиционировать элементы вашей иконки. Действия, что включены также помогут вам генерировать разные размеры, требуемые Apple.

Minimal Icon Generator

Шаблон Minimal Icon Generate довольно гибок. Он содержит ряд векторных элементов с которыми вы можете работать, пачкой градиентов, и выбором цветовых комбинаций. Файлы Photoshop организованы в группы и слои. Если у вас есть хоть немного опыта с Adobe Photoshop, у вас не будет проблем с поиском классной комбинации для вашего приложения.

iOS Bundle: Icon Creator, Generator & Exporter

Если плоский дизайн не то, что вы ищете, тогда этот шаблон iOS-иконки лучший для вашего проекта. Шаблон включает несколько иконок, рамок и задних фонов, которые вы смешивать, пока не найдёте лучшую комбинацию. В шаблоне используются векторы, поэтому у вас не должно возникнуть проблемы с его настройкой в соответствии с вашими потребностями.

Вам нравятся иконки с стиле старой школы, которые использовались в iOS 7? Этот шаблон для вас. Он имеет две дюжины иконок с дизайном реальных материалов, которые вам могут понравится. Также в шаблоне есть градиентные задние фоны, стиля рамок и несколько текстур.

iOS Retina Icon Creator

Этот шаблон объединяет цвета, образцы и шаблоны, предлагающие тысячи комбинаций для создания обычной иконки приложения. Если ваш бюджет ограничен и вы не можете позволить себе дизайнера, тогда это может быть вашим лучшим вариантом. Шаблон включает различные световые эффекты, стили рамок и цветовые комбинации для создания уникальной иконки для вашего приложения.

2. Шаблоны приложения

Шаблоны для приложений значительно ускорит разработку самого приложения. Они помогут придать вашему приложению обаятельный вид и ощущение. GraphicRiver предлагает широкий ряд шаблонов, которые вы можете использовать для своих iOS проектов. Я выбрал несколько, чтобы дать вам представление о том, что доступно на GraphicRiver.

Slade Restaurant iOS Mobile App Template

Этот шаблон приложения сделан на заказ для ресторанов, кафе и прочих мест отдыха. Он поддерживает iPhone 5, iPhone 6 и iPhone 6 Plus. Этот элемент включает в себя множество хорошо организованных по слоям файлов Photoshop. Хотя название ссылается на iOS, этот шаблон также можно использовать для Android.

Concept Keynote Template

Этот шаблон взорвёт ваше сознание. Он разработан для работы с Keynote и имеет поддержку и iPhone и iPad. Шаблон содержит тёмную и светлую версии. Вы можете использовать drag-and-drop для создания макетов, что упрощает создание приложения, которое соответствует вашим потребностям.

iOS App GUI Kit

Несмотря на то, что этот шаблон приложения не является плоским, он включает в себя пять великолепных многоцелевых шаблонов. Файлы Photoshop аккуратно организованы, что позволяет легко выбирать компоненты, которые вы хотите включить в приложение. Каждый шаблон фокусируется на конкретном типе приложения:

  • приложение карт
  • приложение напоминаний
  • чат
  • фотография
  • музыкальный плеер

iOS 9 UI Shop

Этот гламурный, элегантный шаблон демонстрирует мобильное приложение для одежды или магазина одежды. Конструкции детальны, современны и просты в настройке. Шаблон поддерживает дисплеи Retina и хорошо организован.

iOwn UI App

Несмотря на то, что этот плоский шаблон дизайна для iOS является приложением для публикации фильмов, его можно легко модифицировать для других типов приложений. Шаблон включает 12 файлов Photoshop, которые аккуратно организованы и отсортированы по слоям. Само собой разумеется, что шаблон поддерживает устройства с дисплеем Retina.

Minimal App UI Kit

Захватывающей особенностью этого шаблона является его многосторонность. Он демонстрирует широкий ряд типов приложений, что делает его очень полезным для многих проектов. Шаблон имеет две версии,одна из которых для 4-дюймовых дисплеев (iPhone 4, 4S, 5, 5S), а вторая - для 4.7-дюймовых (iPhone 6 и iPhone 6S).

Mobile App Bootstrap UI

Этот шаблон включает более 30 экранов, состоящих из векторных фигур. Элемент включает в себя белый и зеленый варианты, и, кроме всего прочего, он включает более 60 пользовательских векторных значков, которые вы можете использовать в своих приложениях.

Expedition UI Kit

Этот шаблон предназначен для приложений путешествий. Шаблон с не менее чем 30 экранами и десятью категориями стоит своей цены. Он поддерживает iPhone 6 и содержит более чем 100 компонентов.

3. Иконки

Иконки, важная часть мобильного устройства. Экраны мобильных устройств зачастую слишком малы, чтобы содержать много текста, и поэтому иконки лучше подходят для мобильных пользовательских интерфейсов. GraphicRiver предлагает очень много наборов иконок. Я уверен вы можете найти набор, что идеально подойдёт вашему проекту.

Flat UI Icon Set

Большинство мобильных устройств имеют небольшие размеры экрана, поэтому значки - лучший способ дать пользователю интерфейс вашего приложения. Хорошо спроектированные значки помогают пользователю выяснить, как приложение должно использоваться.

Этот набор значков отлично подходит для плоских проектов. Он включает в себя 50 многофункциональных значков, которые можно использовать в любом мобильном приложении.

Thin Weather Icon Set

Его значки - один из основных элементов любого приложения погоды. Этот набор иконок включает 20 гладких высококачественных значков для каждого типа погоды.

Sports Vector Icons

Этот набор значков сфокусирован на спорт и включает в себя не менее 150 векторных иконок. Каждый популярный вид спорта включен в этот набор. Значки детализированы и подходят для iOS, Android и Интернета.

300 iOS and Android Vector Icons

Этот набор включает не менее 300 иконок. Значки отлично подходят для любого проекта для iOS, который имеет плоский дизайн,а также вы можете использовать их для Android или веб-проектов.

Если вы ищете немного больше цвета, вам может понравиться данный набор иконок. Набор включает 80 плоских иконок в разных форматов. Иконки варьируются от мультимедиа и торговли до программирования и образования.

Flat Linear Icon Set

Автор этого набора вышел за все рамки. Элемент содержит не менее 600 значков, включая файлы Photoshop и Illustrator. Некоторые наборы включают погоду, мультимедиа, продукты питания, социальные сети и здоровье.

Web and Mobile Vector Icons

Что мне нравится в этом наборе иконок, так это простота, стиль и цветовые комбинации. Этот набор включает чуть более 150 многофункциональных значков, которые подходят практически для любого проекта. Значки доступны в нескольких форматах и разрешениях.

Сделайте свой выбор

GraphicRiver предлагает феноменальный выбор иконок и шаблонов. Если вы не найдете то, что вам надо, вы, вероятно, не найдете это нигде. В следующий раз, когда вам понадобится графический ресурс для проекта, взгляните на GraphicRiver.
