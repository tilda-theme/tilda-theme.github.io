Создаём Изометрический Писель Арт Комнаты в Adobe Photoshop...

Design & Illustration Pixel Art
Tutorial

Создайте свой собственный дом! В этом уроке я покажу вам, как создать изометрический пиксель арт комнаты.

Получите ещё больше потрясающих вдохновений в стиле Пиксель Арт для своих работ на сайте GraphicRiver.

Исходные материалы

Следующие исходные материалыбыли использованы для создания данного урока:

  • комната.zip

Введение

Перед тем, как мы приступим к уроку, скачайте исходный zip файл, чтобы установить узор. Для этого идём Редактирование - Наборы - Управление наборами (Edit > Presets > Preset Manager). Поменяйте Тип набора (Preset Type) на Узоры (Patterns).

После этого, нажмите кнопку Загрузить (Load) и в появившемся окне браузера, выберите файл с узором, где вы его сохранили, а затем нажмите кнопку Загрузить (Load).

Не забудьте Сохранить (Save) изображения на свой компьютер. Это поможет вам создать тот же самый результат.

1. Создаём Основу Комнаты

Шаг 1

Нажмите клавишу (D), чтобы выставить цвета переднего и заднего плана по умолчанию. Имейте в виду, что для всех инструментов, которые мы будем использовать в этом уроке, следует отключить функцию Сглаживание (Anti-alias).

Откройте программу Photoshop (я использую версию Photoshop CS6), идём Файл - Новый (File > New), установите следующие размеры документа 412 x 395 px. Содержимое Заднего фона (Background Contents) Белый (White).

Шаг 2

Далее идём Слой- Новый - Группа (Layer > New > Group), чтобы создать новую группу. Назовите эту группу Комната (Room). Создайте новый слой внутри созданной группы, для этого идём Слой- Новый - Слой (Layer > New > Layer).

Шаг 3

Выберите инструмент Карандаш (Pencil Tool (B), создайте две L-образные линии, размеры которых составляют 192 x 1 px в обоих направлениях (два центральных пикселя перекрывают друг друга). По два пикселя на каждый нижний пиксель (Прим.переводчика: соотношение ячейки составляет 2:1). Расстояние между линиями составляет 170 px.

Шаг 4

Выберите инструмент Лассо (Lasso Tool (L), отключите опцию Сглаживание (Anti-alias), а затем выделите нижнюю линию. Продублируйте выделенную линию, поверните на 180°. Расположите дубликат линии ниже других линий. Выберите инструмент Карандаш (Pencil Tool (B), чтобы соединить верхние и нижние части. Примечание переводчика: для поворота дубликата линии, активируйте инструмент Свободная трансформация (Free Transform Tool (Ctrl+T).

Шаг 5

Далее, я добавил внешний контур вдоль комнаты, чтобы показать толщину кирпича. Удалите лишние пиксели (которые выделены розовым цветом). Поменяйте название слоя на Комната (Room).

Шаг 6

Выберите инструмент Волшебная палочка (Magic Wand Tool (W), отключите опцию Сглаживание (Anti-alias), а затем щёлкните по полу. Создайте новый слой, залейте тот слой цветом переднего плана. Далее, дважды щёлкните по созданному слою, чтобы добавить стиль слоя Наложение узора (Pattern Overlay). Установите следующие настройки, которые указаны ниже.

  • Режим наложения (Blend Mode): Перекрытие (Overlay)
  • Непрозрачность (Opacity): 100%
  • Щёлкните по миниатюре Узора (Pattern), чтобы выбрать узор напольного покрытия.
  • Масштаб (Scale): 100%

Шаг 7

Возвращаемся к слою Комната (Room). С помощью инструмента Волшебная палочка (Magic Wand Tool (W), отключив опцию Сглаживание (Anti-alias), выделите стены. Создайте новый слой, залейте тот слой цветом переднего плана. Далее, дважды щёлкните по созданному слою, чтобы добавить стиль слоя Наложение узора (Pattern Overlay). Уменьшите непрозрачность Заливки (Fill) слоя до 0%. Установите следующие настройки, которые указаны ниже.

Добавьте стиль слоя Наложение узора (Pattern Overlay) со следующими настройками:

  • Режим наложения (Blend Mode): Перекрытие (Overlay)
  • Непрозрачность (Opacity): 100%
  • Щёлкните по миниатюре Узора (Pattern), чтобы выбрать узор обоев
  • Масштаб (Scale): 100%

Шаг 8

Ещё раз возвращаемся к слою Комната (Room). Заполните все пустоты.

Шаг 9

Выберите инструмент Карандаш (Pencil Tool (B), поменяйте цвет внутренних чёрных линий, используя следующие цветовые оттенки:

  • Пол: #b78c71
  • Светлые блики: #ffffff
  • Стена: #4c848d

Шаг 10

Создайте плинтус, используя следующие цвета: #bbb9b6 и #ffffff. Объедините все слои внутри созданной группы. Временно отключите видимость слоя с группой.

2. Создаём Диван

Шаг 1

Нажмите клавишу (D), чтобы выставить цвета переднего и заднего плана по умолчанию.

Далее идём Слой- Новый - Группа (Layer > New > Group), чтобы создать новую группу. Назовите эту группу Диван (Sofa). Создайте новый слой внутри созданной группы, для этого идём Слой- Новый - Слой (Layer > New > Layer).

Шаг 2

Выберите инструмент Карандаш (Pencil Tool (B), нарисуйте две L-образные линии со следующими размерами 18 x 1 px по ширине и 66 x 1 px в глубину (расстояние между линиями составляет 40 px).

Шаг 3

Выберите инструмент Лассо (Lasso Tool (L), отключите опцию Сглаживание (Anti-alias), а затем выделите верхнюю линию. Продублируйте выделенную линию, поверните на 180°. Расположите дубликат линии поверх других линий. Выберите инструмент Карандаш (Pencil Tool (B), чтобы соединить верхние и нижние части. Удалите лишние пиксели.

Шаг 4

Заполните пустоты между линиями следующими цветовыми оттенками: #f8f8c6, #d8d0a0 и #7f7757.

Шаг 5

Поменяйте цвет светового блика вдоль кромки на #f8f8e8. Установите цвет переднего плана на #4e4135. После этого, выберите инструмент Заливка (Paint Bucket Tool (G), отключите опцию Смежные пиксели (Contiguous), а затем щёлкните по чёрному внешнему контуру. Правый подлокотник готов.

Шаг 6

Продублируйте правый подлокотник, сместите дубликат слоя влево на 112 px (и вверх на 56 px).

Шаг 7

Используя ту же самую технику, цвета и инструменты, создайте другие прямоугольные элементы для нашего дивана (каждый элемент дивана на отдельном слое).

Шаг 8

Объедините все прямоугольные элементы, чтобы получить наш диван. Временно отключите видимость слоя с группой.

3. Создаём Ковёр

Шаг 1

Перед тем, как продолжить, нажмите клавишу (D), чтобы выставить цвета переднего и заднего плана по умолчанию.

Далее идём Слой- Новый - Группа (Layer > New > Group), чтобы создать новую группу. Назовите эту группу Ковёр (Carpet). Создайте новый слой внутри созданной группы, для этого идём Слой- Новый - Слой (Layer > New > Layer).

Шаг 2

Нарисуйте линии со следующими размерами 146 x 1 px в ширину и 138 x 1 px в глубину (первые два пиксели каждой линии перекрывают друг друга).

Шаг 3

Продублируйте слой, сместив дубликат слоя на 10 рх. Повторите действие. В результате вы получите три слоя с L-образными линиями. Объедините все эти слои. Удалите лишние пиксели (которые выделены розовым цветом).

Шаг 4

Давайте добавим верхние линии к ковру. Идём Слой- Создать дубликат слоя (Layer > Duplicate Layer), чтобы продублировать объединённый слой с тремя линиями. Далее, идём Изображение - Вращение изображения - 180° (Image > Image Rotation > 180°), чтобы повернуть дубликат слоя. Расположите дубликат слоя над другими линиями. Объедините оба слоя.

Шаг 5

Выберите инструмент Заливка (Paint Bucket Tool (G), залейте пустоты между линиями, используя следующие цветовые оттенки #73719c и #b5b2de. Установите цвет переднего плана на #504f76, а затем щёлкните по чёрным линиям. Временно отключите видимость слоя с группой.

4. Создаём Стол

Шаг 1

Перед тем, как продолжить, нажмите клавишу (D), чтобы выставить цвета переднего и заднего плана по умолчанию.

Далее идём Слой- Новый - Группа (Layer > New > Group), чтобы создать новую группу. Назовите эту группу Стол (Table). Создайте новый слой внутри созданной группы, для этого идём Слой- Новый - Слой (Layer > New > Layer).

Шаг 2

Я добавил две линии со следующими размерами 62 x 1 px (по два пикселя на каждый нижний пиксель). Это будет ширина нашего стола.

Шаг 3

А теперь добавьте немного глубины.

Шаг 4

Давайте добавим верхние линии к нашему столу. Перейдите на слой с L-образной линией, чтобы продублировать её (Ctrl+J). Далее идём Изображение - Вращение изображения - 180° (Image > Image Rotation > 180°), чтобы повернуть дубликат слоя. Объедините слои, а затем выполните заливку, используя цветовой оттенок #dfbf8e.

Шаг 5

Идём Слой- Создать дубликат слоя (Layer > Duplicate Layer), чтобы продублировать объединённый слой. Перейдите на оригинальный слой, чтобы сместить его на 4 px. Соедините их с помощью инструмента Карандаш (Pencil Tool (B). Объедините все слои внутри группы Стол (Table).

Шаг 6

С помощью инструмента Заливка (Paint Bucket Tool (G), залейте переднюю и боковые стороны следующими цветовыми оттенками: #c9a979 (светлее) и #95675a (темнее). Выберите инструмент Ластик (Eraser Tool (E), чтобы удалить лишние пиксели (я выделил лишние пиксели розовым маркером).

Шаг 7

Установите цвет переднего плана на #f4e1c3, а затем добавьте световую кромку с помощью инструмента Карандаш (Pencil Tool (B). Поменяйте цвет переднего плана на #2b2b2b. Выберите инструмент Заливка (Paint Bucket Tool (G), отключите опцию Смежные пиксели (Contiguous), а затем щёлкните по чёрной линии.

Шаг 8

Создайте новый слой, расположите его ниже всех слоёв внутри группы Стол (Table). Нарисуйте ножку стола (используйте цвет #404968 для внешнего контура, #6b949c для тёмной стороны и #c5eaf0 для блика).

Шаг 9

Продублируйте слой три раза, сместите дубликаты слоёв на их места. Объедините все слои. Временно отключите видимость слоя с группой.

5. Создаём Телевизор

Шаг 1

Перед тем, как продолжить, нажмите клавишу (D), чтобы выставить цвета переднего и заднего плана по умолчанию.

Далее идём Слой- Новый - Группа (Layer > New > Group), чтобы создать новую группу. Назовите эту группу ТВ (TV). Выберите самый тонкий прямоугольник из конструкции дивана, а затем продублируйте его. Переместите дубликат прямоугольника в группу ТВ (TV). Выберите инструмент Прямоугольная область (Rectangular Marquee Tool (M), выделите нижнюю часть прямоугольника. Щёлкните правой кнопкой по обрасти выделения и в появившемся окне, выберите опцию Вырезать на новый слой (Layer via Cut). Сместите созданный новый слой вниз на 16 px. Объедините слои внутри группы. Соедините линиями, а также заполните пустоты.

Шаг 2

Поменяйте цвета:

  • Верхняя плоскость: #6e7488
  • Передняя плоскость: #525663
  • Боковая плоскость: #383c45
  • Световые блики: #818796
  • Внешний контур: #20232e

Шаг 3

Выберите инструмент Карандаш (Pencil Tool (B), нарисуйте экран. Залейте экран следующим оттенком #000000.

Шаг 4

Установите цвет переднего плана на #000000, а затем нарисуйте динамики. Объедините все слои внутри группы.

Поздравляю, мы завершили урок!

Включите видимость все слоёв с группами. Далее, идём Файл- Поместить (File > Place), чтобы поместить дополнительные элементы (картины, цветочный горшок). Рассортируйте слои и группы. Разместите элементы интерьера на своё усмотрение. Вы можете увидеть мой вариант, как расположить все предметы.

На скриншоте ниже вы можете увидеть итоговый результат (100%). Надеюсь, что вам понравился этот урок и не забудьте поделиться с нами своим результатом!
