Создаём Графический Фото Эффект в Стиле 90х в Adobe Photoshop...

Design & Illustration Photo Effects
Tutorial

Девяностыебыли наполнены невероятными, яркими цветами и абстрактными формами! И вы можетесоздать потрясающий фотоэффект в стиле 90х, используя пользовательские фигуры имногое другое, в Adobe Photoshop.

В этомуроке вы узнаете, как вначале создать задний фон, а затем добавить фигуры, модельи заключительные цвета к нашей композиции.

Хотитеещё больше идей? Создайте красивый ретро фото эффект с помощью Экшена Photoshop от Envato Elements.

Давайтеприступим!

Исходные материалы:

Следующиеисходные материалы были использованы для создания данного урока:

  • Модель

МузыкальноеВидео для Вдохновения

Одно изсамых лучших мест для поиска вдохновения – это ваше любимое музыкальное видео.Этот урок вдохновлён крутой энергией музыкального видео из 90х: Hard Times by Paramore.

Вместосоздания точной копии оригинала, мы просто примем во внимание различные элементы дизайна, которыемы увидим здесь. Они помогут нам создать свой собственный дизайн!

Поэтомуя сосредоточусь на добавлении простых форм и ярких красок для достижения этогоэффекта.

1. Создаём Задний Фон в Стиле 90х

Шаг 1

Создайтеновый документ со следующими размерами 1000 x 648 px. Залейтезадний фон тёмно-фиолетовым цветом: #211f26 спомощью инструмента Заливка (Paint Bucket Tool (G).

Шаг 2

Щёлкнитеправой кнопкой мыши по фоновому слою и в появившемся окне, выберите опцию Параметрыналожения (Blending Options). Добавьте стиль слоя Наложение узора (Pattern Overlay), выберите узор Right Diagonal Line, а также установите следующие настройки, которые указанына скриншоте ниже:

Результатдолжен быть, как на скриншоте ниже. Задний фон выглядит вначале тёмным, но мы осветлимего в дальнейших шагах.

Шаг 3

Создайтеновый слой поверх фонового слоя. С помощью мягкой круглой кисти Жёсткостью (Hardness) 0%, добавьтебольшой пятно синего цвета: #2e6be1 в верхнем левом углу нашей композиции.

2. Добавляем Первые Абстрактные Фигуры

Шаг 1

Перед тем,как мы добавим модель, давайте сфокусируемся на создании первых абстрактных фигурдля нашей сцены. Для этого, нам нужно добавить фигуры Все (all) черезменю инструмента Произвольная фигура (Custom Shape Tool (U).

Выберитеинструмент Произвольная фигура (Custom Shape Tool (U), а затем в выпадающем меню, выберите Все (All). Когдавам будет предложено заменить / добавить, нажмите кнопку Добавить (Append).

Теперьу нас есть все фигуры, необходимые для этого урока!

Шаг 2

Выберитеинструмент Эллипс (Ellipse Tool (U), добавьте большой синий круг #0467ea, слегкасместив его от центра.

Шаг 3

Добавьтенесколько фигур к созданному кругу, это сделает его более выделяющимся! Создайтесвою фигуру, используя инструмент Линия (Line Tool (L).

Создайте зигзагообразную фигуру со следующими значениями:

  • Толщина(Weight): 4 px
  • Цвет (Color):#002ad7
  1. Вначаленарисуйте одну линию, далее, нажмите клавиши (Ctrl+J),чтобы продублировать эту линию.
  2. Повернитедубликат слоя с помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T) таким образом, чтобы дубликат линии был расположенперпендикулярно первой линии.
  3. Продолжайтепроцесс соединения линий вместе, пока вы не получите фигуру, как на скриншотевыше.

Когда вызакончите, выделите все элементы фигуры, а затем щёлкните правой кнопкой мыши повыделенным слоям и в появившемся окне, выберите опцию Объединить фигуры (Merge Shapes).

Шаг 4

Нажмитеклавиши (Ctrl+J), чтобы создать ещё больше дубликатов слоёв с зигзагообразнымифигурами. Расположите зигзагообразные фигуры по всему кругу. Далее, выделите всеслои с этими фигурами, а затем щёлкните правой кнопкой мыши по выделенным слоями в появившемся окне, выберите опцию Создать обтравочную маску (Create a Clipping Mask) к слою с кругом.

Теперь создайте тонкую чёрную линию Толщиной (Weight) 2рх. Дважды продублируйте линию (Ctrl+J), чтобы получить три чёрные линии на круге. Преобразуйте фигуры в обтравочные маски к слою с эллипсом.

Шаг 5

Добавьтебольше абстрактных фигур! Используя инструмент Произвольная фигура (Custom Shape Tool (U), выберите фигуру треугольника из меню фигур, чтобы добавитьнесколько красочных треугольников к кругу.

Яиспользовала следующие цвета:

  • Зелёный:#39b54a
  • Жёлтый: #fff568
  • Розовый: #f06eaa

Добавьтекрошечные кружочки белого цвета в нашу смесь фигур с помощью инструмента Эллипс(Ellipse Tool (U).

Ещё развыберите инструмент Произвольная фигура (Custom Shape Tool (U), чтобы добавить несколько белых кружков.

Далее,добавьте крестики. Чтобы создать их, просто нарисуйте белую линию толщиной 1px, азатем продублируйте её (Ctrl+J), поверните дубликат слоя, а затем объедините фигуры.Добавьте столько крестиков на изображение круга, сколько вы хотите.

3. Добавляем Больше Фигур и Красок

Шаг 1

Чтобысбалансировать фигуры, которые мы создали внутри круга, с остальной частьюкомпозиции, мы добавим несколько фигур на задний план. Но сначала давайте проведёмкоррекцию цвета.

Идём Слой-Новый корректирующий слой- Яркость / Контрастность (Layer > New Adjustment Layer > Brightness & Contrast). Установите следующие настройки, которые указаныниже:

  • Яркость(Brightness): 21
  • Контрастность(Contrast): 25

Шаг 2

Добавьтенесколько больших треугольников тёмно-фиолетового цвета #292631 на задний фон. Спомощью инструмента Свободная Трансформация (Free Transform Tool (Ctrl+T), придайте треугольникам различные углы наклона.

Создайтеновый слой поверх фигур. Далее, с помощью инструмента Кисть (Brush Tool (B) Жёсткостью (Hardness) 0%, добавьтебольшое зелёное пятно #2edfb0 поверх синего пятна в верхнем левом углу. Уменьшитезначение Непрозрачности (Opacity) слоя до 42%.

Небойтесь экспериментировать. Добавьте ещё фигур, это улучшить вашу композицию!

  1. Добавьтетри больших тёмных треугольника. Создайте их с наложением на другие фигуры.
  2. Используйтечёрный и синий цвет #07101c дляпервых двух треугольников.
  3. Далее,создайте третий треугольник синего цвета #242954, а затем добавьте к нему стильслоя Наложение Градиента (Gradient Overlay). Установите следующие настройки, которые указаны наскриншоте ниже:

Добавьтеещё несколько треугольников к нашей композиции, включая треугольники зелёногоцвета #00a651.

Результатдолжен быть, как на скриншоте ниже.

Шаг 3

Для усиленияатмосферы, давайте создадим простую виньетку. Для этого, создайте новый слой поверхвсех остальных слоёв.

Выберитеинструмент Градиент (Gradient Tool (G), находясь на новом слое, добавьте Радиальный (Radial)градиент, цвета градиента от чёрного до прозрачного. Не забудьте поставить галочкув окошке Инверсия (Reverse), чтобы получить красивый эффект виньетки.

Далее,добавьте два новых корректирующих слоя. Вначале, добавьте корректирующий слой Поискцвета (Color Lookup). В выпадающем меню 3DLUT File, выберите фильтр 2Strip.look.

Затем добавьтекорректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Средних тонов (Midtones) и Света(Highlights), которые указаны на скриншоте ниже:

Для усиленияглубины, добавьте корректирующий слой Яркость / Контрастность (Brightness and Contrast).

  • Яркость(Brightness): 27
  • Контрастность(Contrast): 54

Теперьдавайте, наконец, добавим модель!

4. Добавляем Модель

Шаг 1

Создаваемыйэффект обрамляет изображение модели, поэтому вы можете использовать своёсобственное изображение или используйте любое другое красивое исходноеизображение, как данное исходное изображение мужчины с очками:

Основнаязадача здесь- это выбрать фотографию в стиле 90х для создания общего эффекта. Изучитестили моды, чтобы создать органичный вид. Например, джинсовая куртка никогда невыходит из моды и отлично подходит для данного хипстеровского стиля 90-х.

Скопируйте/ вклейте всё изображение на наш рабочий документ, расположив её по месту.Далее, добавьте слой-маску и с помощью чёрной кисти, прокрасьте все лишние участки,которые вы хотите скрыть. Поиграйте со степенью жёсткости кисти, постарайтесь сохранитьтолько изображение модели, слегка смягчив изображение модели в нижнем правомуглу.Примечание переводчика: чтобы смягчить изображениемодели в нижнем правом углу, используйте мягкую чёрную кисть с низкойнепрозрачностью, просто пройдите кистью по изображению модели один или двараза.

Шаг 2

Щёлкнитеправой кнопкой мыши по слою с моделью и появившемся окне, выберите опциюПараметры наложения (Blending Options). Добавьте стиль слоя Тень (Drop Shadow). Установите следующие настройки, которые указаны наскриншоте ниже:

Шаг 3

Теперьсоздайте новый слой, поменяйте режим наложения для данного слоя на Жёсткий свет(Hard Light). Преобразуйте данный слой в обтравочную маску к слоюс моделью. С помощью инструмента Кисть (Brush Tool (B), прокрасьте мягкой чёрной кистью изображение модели,чтобы добавить тени. Постарайтесь смягчить насыщенность изображения с помощью мягкойтени. Уменьшите значение Непрозрачности (Opacity) до 34%.

Добавьтеновый корректирующий слой Поиск цвета (Color Lookup) в качестве обтравочной маски к слою с моделью. Нам необходимопровести цветовую коррекцию изображения модели, чтобы совместить с задним фоном.

Извыпадающего меню 3D LUT File, выберите фильтр FoggyNight.3DL, азатем поменяйте режим наложения для данного корректирующего слоя на Яркость (Luminosity).

То же самое,что мы проделывали ранее, добавьте новый корректирующий слой в качестве обтравочноймаски, на этот раз добавьте корректирующий слой Яркость / Контрастность (Brightness/Contrast).

  • Яркость(Brightness): -1
  • Контрастность(Contrast): 12

5. Добавляем Заключительные Эффекты

Теперь,когда мы поместили основную часть эффектов, всё, что нам нужно сделать, это добавитьещё несколько эффектов для улучшения композиции.

Шаг 1

Создайтеновый слой, а затем поменяйте режим наложения для данного слоя на Разница (Difference). С помощьюинструмента Кисть (Brush Tool (B), цвет кисти синий #202e4d,прокрасьте мягкой кистью правую сторону композиции, чтобы добавить тени. Слегкаслейте модель с задним фоном.

Шаг 2

Давайтезатемним очки!

Создайтеновый слой, а затем с помощью инструмента Перо (Pen Tool (P), создайте выделенную область вокруг линз, чтобызалить чёрным цветом.

Мы используемчёрный цвет в качестве основы, чтобы добавить больше красок. Создайте нескольконовых слоёв в качестве обтравочных масок к слою с чёрными линзами.

На каждомслое, используйте инструмент Перо (Pen Tool (P), чтобы создать радужные блики на линзах. Используйтеразличные фигуры и цвета на своё усмотрение.

Яиспользовала следующие цвета:

  • Розовый:#fe4b86
  • Фиолетовый:#c813af
  • Синий:#5b6090

Чтобы линзывыглядели более реалистично, давайте добавим к ним мягкие тени! Создайте новый слойв качестве обтравочной маски к слою с чёрными линзами. Поменяйте режим наложениядля данного слоя на Умножение (Multiply), азатем с помощью мягкой чёрной кисти Жёсткостью (Hardness) 0%, прокрасьтекрая очков, чтобы добавить мягкие тени. Смягчите жёсткие линии для созданиякрасивого и естественного эффекта.

Шаг 3

Для созданияэффекта обесточенных линий, используйте Перо (Pen Tool (P), чтобы создать контурные линии вокруг изображения модели(не переживайте слишком сильно насчёт чёткости линий— чем сумасшедшее, темлучше).

Далее,щёлкните правой кнопкой мыши по созданному контуру и в появившемся окне,выберите опцию Выполнить обводку контура (Stroke Path) (не забудьте установите размер инструмента Карандаш(Pencil Tool) на 2 px). Выполните обводку контура, выбрав инструмент Карандаш(Pencil Tool) в выпадающем меню, а также уберите галочку в окошке Имитировать нажим (Simulate Pressure).

В этомшаге вы можете использовать белый или любой другой цвет в качестве цвета обводки.

Продублируйте(Ctrl+J) несколько раз слой с созданной линией. Для каждого новогодубликата слоя с линией, отредактируйте форму и размер линии с помощьюинструмента Свободная трансформация (Free Transform Tool (Ctrl+T). Растяните их в порядке возрастания. Примечаниепереводчика: 1. каждая последующая линия должна быть больше предыдущей поразмеру 2. Количество линий не ограничено, используйте столько линий, сколькопожелаете.

Далее,щёлкните правой кнопкой мыши по каждому слою с линией и появившемся окне,выберите опцию Параметры наложения (Blending Options).

Применитестиль слоя Наложение цвета (Color Overlay), чтобы применить различные цветовые оттенки к каждойлинии.

Яиспользовала следующие цвета:

  • Розовый:#ffa5c5
  • Жёлтый:#fff7b4
  • Фиолетовый:#242645

Результатдолжен быть, как на скриншоте ниже.

Шаг 4

Создайтеновый слой поверх всех остальных слоёв, поменяйте режим наложения для данного слояна Перекрытие (Overlay). С помощью мягкой кисти, цвет кисти белый, прокрасьтеповерх грудной клетки мужчины, слегка добавляя свет в эту область. Уменьшитезначение Непрозрачности (Opacity) до 65%.

Далее,добавьте два корректирующих слоя для заключительной коррекции цвета.

Вначаледобавьте корректирующий слой Цветовой Баланс (Color Balance). Установите следующие настройки для Средних тонов (Midtones) иСвета (Highlights), которые указаны на скриншоте ниже.

Затем добавьтевторой корректирующий слой Поиск цвета (Color Lookup), а затем в выпадающем меню 3D LUT File, выберите фильтр 2Strip.look. УменьшитеНепрозрачность (Opacity) слоя до 45% для создания бриллиантового -голубоговида.

Наэтом всё! Результат должен быть, как на скриншоте ниже.

Попробуйтеприменить этот же эффект на других фотографиях для получения более потрясающихрезультатов!

Заключение

Отпразднуйтеклассические стили 90-х годов с этим потрясающим фото эффектом!

В этомуроке вы узнали, что несколько хорошо расположенных фигур могут подчеркнуть любуюинтересную исходную фотографию. Не забудьте установить пресеты для доступа кудивительным векторным фигурам и т.д. и не бойтесь поэкспериментировать с корректирующимислоями, чтобы настроить цветовую схему на своё усмотрение!
