Как создать эффект калейдоскопа в Adobe Photoshop...

Design & Illustration Music Video
Tutorial

Поскольку мы продолжаем нашу серию по эффектам, на которые вдохновили музыкальные видео, одним из эффектов, которыми я всегда восхищался, был эффект калейдоскопа. Лучший пример этого эффекта можно увидеть в визуально богатом видео Pillow Talk от Zayn. Посмотрите его на 1:20, 1:55, 2:26 и 3:12.

Задача состоит в том, чтобы легко найти способ создания эффекта, а не вручную размещать десятки копий.

К счастью, Photoshop имеет несколько полезных функций, чтобы облегчить такую задачу. Во-первых, это часто забытая Scripted Fill, которая делает короткую работу по созданию калейдоскопических узоров и имеет достаточно возможностей, чтобы вы были заняты экспериментированием в течение нескольких часов! Сопряженная с ней функция Transform Again является интересным способом создания повторяющихся объектов, похожих на способ работы старого спирографа.

1. Создайте паттерн

Scripted fill очень полезен для этого типа эффекта, но он должен использовать предопределенный шаблон для работы. Паттерны, которые не являются квадратными, имеют тенденцию работать лучше всего, поэтому мы вырежем музыкального исполнителя из фона и создадим шаблона для использования.

Шаг 1

Этот метод будет работать практически с любым изображением, но для этого проекта давайте работать с фотографией музыканта, чтобы лучше всего имитировать исходный эффект из видео. В частности, мы будем использовать это изображение красивой девушки, которая играет на гитаре с Envato Market.

Шаг 2

Возьмите Quick Selection Tool (W) и нарисуйте вдоль модели, чтобы создать выделение, которое полностью окружит ее. Если инструмент захватывает область фона, удерживайте клавишу Alt и красьте вдоль фона, чтобы снять выделение.

Шаг 3

Нажмите кнопку Refine Edge для запуска диалогового окна. Отрегулируйте Radius до 2 px и выберите функцию Smart Radius. Затем используйте инструмент Refine Radius Tool, чтобы покрасить по краям волос модели для создания лучшего выделения на этих длинных локонах. Затем нажмите кнопку ОК, чтобы вернуться в основной интерфейс Photoshop.

Шаг 4

Используйте выделенную область как маску слоя, перейдя в Layer > Layer Mask > Reveal Selection.

Шаг 5

Нажмите на миниатюру слоя, чтобы убедиться, что фокус находится на слое, а не на маске слоя. Затем перейдите в Edit > Define Pattern. Назовите паттерн Guitarist.

2. Scripted Fill легко

С помощью нашего пользовательского паттерна, определенного и готового к работе, пришло время использовать Scripted Fill для создания узора калейдоскопа. Существует несколько вариантов, с которыми можно экспериментировать; перечисленные здесь - всего лишь предложение начать работу. Не стесняйтесь выработать свой собственный эффект узора.

Шаг 1

Создайте новый документ в File > New. Задайте и Width и Height на 2500 px. Задайте Background Contents на Other и укажите черный цвет (#000000).

Шаг 2

Создайте новый слой в Layer > New > Layer (Shift-Ctrl-N) и назовите новый слой Scripted Fill.

Шаг 3

Перейдите в Edit > Fill и измените Contents на Pattern и выберите Custom Pattern на паттерн Guitarist, который вы создали ранее. Затем установите флажок на Script и выберите Symmetry Fill.

В диалоговом окне Symmetry Fill выберите следующие параметры:

  • Symmetry type: 32 wallpaperP6 symmetry
  • Pattern Scale: 0.75
  • Pattern translation along width: 25% от ширины
  • Pattern translation along height: -40% от высоты

Шаг 4

Добавьте корректирующий слой Hue/Saturation и выберите следующие параметры:

  • Hue: 0
  • Saturation: -57
  • Lightness: -62

Шаг 5

Используйте инструмент Ellipse Tool (U), установленный в Path, и нарисуйте круг, расположенный по центру холста. Удерживание нажатой клавиши Shift во время рисования круга держит форму, как идеальный круг.

Шаг 6

Добавьте новый слой, названный Guitar Circle и перейдите в Edit > Fill. Измените сценарий заливки на Place Along Path. Затем в диалоговом окне Place Along Path, примените следующие параметры:

  • Pattern Scale до 0,45
  • Spacing до -500 пикселей
  • Проверьте настройки Adjust spacing to fit
  • Distance from path до 295 пикселей
  • Scale progression до 100%
  • Color и Brightness randomness до 0

3. Преобразование ярлыков

Чтобы создать эффект «цветения», мы будем использовать функцию Transform Again. Эта функция больше похожа на счастливую случайность, чем на предполагаемую функцию. Независимо от того, почему это так в программе, это, безусловно, полезно для быстрого создания геометрических шаблонов!

Шаг 1

Удерживайте нажатой клавишу Alt и перейдите в Edit > Transform (Ctrl-T). Это создаст копию слоя Guitar circle. В панеле параметров задайте height и width масштаба на 90% и rotation на 20 градусов.

Шаг 2

Удерживайте нажатыми клавиши Alt-Shift-Ctrl и нажмите клавишу T. Это команда Transform Again которую вы можете найти в меню Edit > Transform. Это сделает еще одну копию слоя и применит предыдущее трансформирование. Продолжайте нажимать ту же комбинацию клавиш, пока не будет пять копий слоя Guitar Circle.

Шаг 3

Добавьте корректирующий слой Hue/Saturation поверх слоя Guitar Circle copy 2. Закрепите его на этом слое с помощью Layer > Create Clipping Mask (Alt-Ctrl-G). Затем примените следующие параметры для корректирующего слоя:

  • Hue: 0
  • Saturation: -22
  • Lightness: -56

Шаг 4

Добавьте другой корректирующий слой Hue/Saturation над слоем Guitar Circle copy 2. Закрепите его на этом слое с помощью Layer > Create Clipping Mask (Alt-Ctrl-G). Затем примените следующие параметры для корректирующего слоя:

  • Hue: 0
  • Saturation: -22
  • Lightness: + 7

Шаг 5

Добавьте новый слой с именем Gradient в верхней части стека слоев. Убедитесь, что цвет переднего плана foreground установлен на black, и выделите инструмент Gradient Tool (G). Используйте Foreground to Transparent и градиент фигуры Radial. Затем нарисуйте градиент, начиная с центра полотна.

4. Закончите с помощью некоторых линий

Заключительный этап - использование тех же самых методов, но и раньше, но добавим некоторые простые линии и формы, чтобы объединить весь эффект во что-то особенное.

Шаг 1

Используйте инструмент Polygon Tool (U) и установите Sides на 5, Fill на null, Stroke Color на white и stroke до 4 пикселей. Затем нарисуйте пять-многоугольника в центре холста.

Шаг 2

Перейдите в Layer > Rasterize Shape, чтобы превратить векторную фигуру в пиксельную. Затем удерживая клавишу Alt, одновременно нажимайте на visibility toggle icon (это значок глазного яблока) для слоя Polygon. Это сделает только этот единственный слой видимым. Перейдите в Edit > Define Pattern и назовите этот паттерн Polygon.

Шаг 3

Удерживая клавишу Alt, нажмите на значок visibility toggle icon слоя Polygon, чтобы снова отобразить все остальные слои. Добавьте новый слой new layer поверх слоя Hue/Saturation 1 и назовите его фоновые линии Background Lines.

Шаг 4

Используйте Edit > Fill снова и задайте Custom Pattern паттерну Polygon, который вы только что создали. Также установите Script на Symmetry Fill.

В диалоговом окне Symmetry Fill задайте следующие параметры:

  • Symmetry Type: 33 wallpaperP6M symmetry
  • Pattern Scale: 0.4
  • Pattern translation along width: 50% от ширины
  • Pattern translation along height: 50% от высоты
  • Color randomness: 0
  • Brightness randomness: 0

Затем уменьшите Opacity слоя до 25%.

Шаг 5

Скройте Hide слой Polygon и снова используйте инструмент Polygon Tool (U). Установите Sides до 3 и Stroke до 5 пикселей. Затем нарисуйте треугольник в центре холста.

Шаг 6

Удерживая клавишу Alt, перейдите в Edit > Free Transform. Это создаст новую копию слоя треугольника, чтобы его трансформировать. Rotate приблизительно на 20 градусов.

Шаг 7

Используйте комбинацию клавиш для Transform Again (Alt-Shift-Ctrl-T) примерно 15 раз, чтобы создать эффект эффекта спирографа в центре изображения.

Шаг 8

Выделите все слои треугольников на панели Layers, нажав Shift на первый и на последний. Затем сгруппируйте их с помощью Layer > Group Layers (Ctrl-G) и назовите группу Triangles.

Шаг 9

Перейдите в Layer> Layer Mask> Hide All, чтобы добавить маску слоя в группу. Затем установите цвет переднего плана foreground на white и используйте инструмент Gradient Tool (G). Используйте пресет Foreground to Transparent и Radial для градиента фигуры. Нарисуйте градиент в центре полотна на маске группы, чтобы перекрыть изображение гитаристки треугольниками.

Шаг 10

Используйте инструмент Crop Tool (C), чтобы обрезать изображение чуть ниже центра, чтобы завершить эффект.
