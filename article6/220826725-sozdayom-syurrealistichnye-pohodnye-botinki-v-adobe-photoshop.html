Создаём Сюрреалистичные Походные Ботинки в Adobe Photoshop ...

Design & Illustration Photo Manipulation
Tutorial

Вы любите походы? Обновите свою обувь, заменив её на уникальную пару ботинок в виде ног! В этом уроке, вы узнаете, как с помощью стандартного изображения ног, создать сцену с походными ботинками, используя базовые техники создания фотоманипуляций.

Хотите добавить естественный природный вид ко всем своим фотографиям? Тогда рассмотрите невероятный набор Природных Экшенов Photoshop от GraphicRiver для ретуши своих работ. Или обратитесь за помощью к профессиональным дизайнерам от Envato Studio, если у вас возникли вопросы по фотоманипуляции.

Исходные материалы

Следующие исходные материалыбыли использованы для создания данного урока:

  • Ноги
  • Походные Ботинки
  • Трава
  • Лес
  • Божья коровка

1. Отделяем Ноги

Шаг 1

Для создания композиции, в этом уроке я буду использовать исходные размеры изображения Ног. В данный момент исходные размеры составляют 2200 х 2635 пикселей, но в дальнейшем мы кадрируем данное изображение в соответствии с нашей сценой.

Откройте своё исходное изображение в программе Photoshop. С помощью инструмента Волшебная палочка (Magic Wand Tool (W), выделите белые участки заднего фона, а затем нажмите клавишу (Delete), чтобы удалить их.

Шаг 2

С помощью инструмента Магнитное лассо (Magnetic Lasso Tool (L), создайте выделение вокруг каждой ноги. Нажмите клавиши (Ctrl+J), чтобы продублировать каждую ногу на отдельный слой. Когда вы закончите, то у вас должно быть два отдельных слоя с каждой отдельной ногой. Чтобы сохранить пространство в палитре слоёв, удалите слой с напольными весами, а затем сгруппируйте оба слоя с ногами в одну группу.

2. Рисуем Наброски для Обуви

Шаг 1

С помощью инструмента Перемещение (Move Tool (V), сместите изображение ног немного вниз.

Не бойтесь экспериментировать с расположением ног. Вы можете сместить одну ногу немного назад или пусть они остаются параллельны друг к другу, как это сделала я. Изменение положения ног может просто немного усложнить процесс, так что, имейте это в виду.

Шаг 2

Следующие несколько шагов являются выборочными. Не бойтесь пропустить эти шаги при необходимости, не рисуя наброски. Чтобы мне было легче понять, как расположить обувь, мне нужно нарисовать простой набросок поверх ног. Это поможет мне понять перспективу и где правильно расположить шнурки, крючки и голенище.

Создайте новый слой ниже слоя с группой с ногами, а затем залейте этот слой белым цветом с помощью инструмента Заливка (Paint Bucket Tool (G). Выберите стандартную жёсткую кисть, чтобы нарисовать два круга выше голени на отдельном новом слое.

Объедините слои с ногами. Добавьте слой-маску к объединённому слою, а затем прокрасьте чёрной кистью на слой-маске, чтобы скрыть верхние половины ног. Проделайте это для каждой ноги. Примечание переводчика: 1. щёлкните правой кнопкой мыши по слою с группой с ногами и в появившемся окне, выберите опцию Объединить группу (Merge Group) 2. создайте резервную копию с группой на всякий случай.

Шаг 3

Ноги выглядят немного тонкими. Выделите каждую ногу с помощью инструмента Прямоугольная область (Rectangular Marquee Tool (M), а затем с помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T), увеличьте их. Проведите коррекцию маски и наброска соответственно.

Шаг 4

На слое с наброском, нарисуйте основные контурные линии язычка ботинок. Это даст нам представление, где в дальнейшем расположить исходное изображение с ботинками.

3. Создаём Эффект Ботинок

Шаг 1

Теперь, когда у нас есть хороший ориентир, мы можем добавить ботинки. Во-первых, откройте исходное изображение с походными ботинками в отдельном документе. С помощью инструмента Волшебная палочка (Magic Wand Tool (W), выделите задний фон, а затем нажмите клавишу (Delete), чтобы удалить белый задний фон.

Шаг 2

С помощью инструмента Прямолинейное лассо (Polygonal Lasso Tool (L), создайте выделение вокруг шнурков, языка и голенища левого ботинка. Скопируйте / вклейте выделенное изображение на наш рабочий документ. Слегка уменьшите Непрозрачность (Opacity) слоя так, чтобы вы могли видеть ноги под изображением ботинка. Примечание переводчика: после коррекции, не забудьте вновь увеличить непрозрачность слоя.

Расположите ботинок поверх левой ноги. Примените Масштабирование и Деформацию с помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T), чтобы ботинок подошёл к ноге. Попеременно отключайте/ включайте видимость слоя с наброском, чтобы он не отвлекал ваше внимание.

Продолжайте коррекцию обуви, пока вас не устроит, как ботинок подошёл к ноге. К слою с ботинком, добавьте слой-маску и с помощью мягкой и жёсткой кисти чёрного цвета, скройте лишние участки ботинка. Примечание переводчика: не забудьте перейти на слой-маску слоя.

Шаг 3

Давайте перейдём к правой ноге. То же самое, создайте выделение вокруг правого ботинка с помощью инструмента Прямолинейное лассо (Polygonal Lasso Tool (L), а затем скопируйте / вклейте выделенное изображение на новый слой на наш рабочий документ, расположив поверх слоя с правой ногой.

Эта нога немного сложнее, поэтому, я опишу весь процесс. Нажмите клавиши (Ctrl+T) для Свободной трансформации (Free Transform) ботинка, уменьшите масштаб, чтобы ботинок подходил к правой ноге. Постарайтесь подобрать наилучшее положение, сохраняя соотношение размера с левым ботинком.

Удерживая клавишу (Ctrl), потяните за углы контура области трансформации для небольшой деформации перспективы.

Не выходя из режима трансформации, щёлкните правой кнопкой мыши по изображению и в появившемся окне. выберите опцию Деформация (Warp). Деформируйте ботинок под форму ноги. Не переживайте, если ботинок не плотно ляжет на ногу, в противном случае, вы можете слишком сильно деформировать изображение ботинка.

Шаг 4

Правая нога чуть-чуть видна из-за ботинка, поэтому, нам нужно скрыть лишние участки на слой-маске. Итак, перейдите на слой-маску слоя с правым ботинком и с помощью чёрной кисти, скройте лишние участки ноги.

Таким же образом, скройте любые лишние участки ботинка, которые вам не нравятся. Ели хотите, то вы можете исправить края, для этого создайте новый слой и с помощью кисти, прокрасьте края.

4. Создаём Атмосферу Леса

Шаг 1

Теперь, когда самые сложные шаги позади, мы можем сосредоточиться над созданием лесной сцены. Удалите слой с белым задним фоном. Скопируйте / вклейте исходное изображение Травы на наш рабочий документ, расположив слой с текстурой травы ниже слоя с ботинками. С помощью инструмента Свободная трансформация (Free Transform Tool (Control-T), примените режимы Масштабирование и Перспектива, чтобы увеличить текстуру травы.

Добавьте новый корректирующий слой Цветовой тон / Насыщенность (Hue and Saturation) в качестве обтравочной маски к слою с травой. Обесцветьте траву, чтобы она подошла к ногам. Установите следующие настройки:

  • Цветовой тон (Hue): 0
  • Насыщенность (Saturation): -67
  • Яркость (Lightness): -12

Шаг 2

Объедините слои с ботинками и ногами вместе. Далее, продублируйте объединённый слой (Ctrl+J), слегка сдвинув дубликат слоя.

Далее, идём Изображение - Коррекция - Цветовой тон / Насыщенность (Image > Adjustments > Hue and Saturation). Уменьшите значение Яркости (Lightness) до 0 для дубликата слоя, чтобы затемнить изображение.

Далее, размойте тень, для этого идём Фильтр - Размытие - Размытие по Гауссу (Filter > Blur > Gaussian Blur), установите радиус размытия 48.4 px. Уменьшите значение Непрозрачности (Opacity) слоя с тенью, примерно, до 54%.

Шаг 3

Давайте добавим немного реалистичности к траве. К слою с ботинками, добавьте слой-маску. С помощью круглой жёсткой кисти чёрного цвета, прокрасьте отдельные участки ног, чтобы была видна трава. Примечание переводчика: 1. в качестве кисти вы можете использовать стандартную кисть Трава 2. автор создаёт ощущение, как будто ботинки стоят среди травы 3. не забудьте перейти на слой-маску для коррекции кистью.

Шаг 4

Теперь скопируйте / вклейте исходное изображение с Лесом на наш рабочий документ, чтобы добавить лесную растительность к нашей сцене. Примечание переводчика: расположите слой с кустарником ниже слоя с тенью.

Продублируйте слой с кустарником (Ctrl+J), а затем отразите по горизонтали, для этого идём Редактирование - Трансформация - Отразить по горизонтали (Edit > Transform > Flip Horizontal). Расположите дубликат слоя с кустарником в правой верхней части нашей сцены.

К каждому слою с кустарником, добавьте слой-маску. С помощью мягкой круглой кисти чёрного цвета, прокрасьте кустарники на слой-маске, чтобы скрыть любые жёсткие линии. Постарайтесь, как можно лучше совместить кустарники с травой.

Шаг 5

Кустарники слишком яркие на фоне нашей сцены, поэтому, добавьте новый корректирующий слой Цветовой тон / Насыщенность (Hue and Saturation) в качестве обтравочной маски к каждому слою с кустарником. Уменьшите значение Яркости (Lightness) до -16.

Шаг 6

Перед тем, как мы перейдём к заключительным деталям, давайте изменим цвет голенища обоих ботинков. Вначале, создайте выделение вокруг голенища ботинка с помощью инструмента Прямолинейное лассо (Polygonal Lasso Tool (L). Далее, идём Слой- Новый корректирующий слой - Цветовой тон / Насыщенность (Layer > New Adjustment Layer > Hue and Saturation), чтобы применить слой-маску только к выделенной области голенища.

Поменяйте цвет от оранжевого на фиолетово-синий оттенок, изменив значение Цветового тона (Hue) на -108.

Повторите весь процесс для левой ноги.

5. Добавляем Детали и Эффекты Освещения

Шаг 1

Теперь, когда у нас есть ботинки и лесной фон с травой, мы можем перейти к заключительным деталям, чтобы сделать нашу сцену более выразительной! Вначале добавьте новый корректирующий слой Кривые (Curves) в качестве обтравочной маски к слою с ботинками. Слегка сдвиньте кривую для RGB Синего (RGB Blue) канала.

Откройте исходное изображение Божьей коровки. Отделите божью коровку от заднего фона, а затем скопируйте / вклейте данное изображение на наш рабочий документ. Поместите божью коровку на одной из ног, чтобы создать причудливый эффект.

Шаг 2

Теперь нам нужно поработать над освещением и общей цветовой схемой. Создайте новый слой, поменяйте режим наложения для данного слоя на Умножение (Multiply), а также уменьшите значение Непрозрачности (Opacity) слоя до 55%. С помощью большой мягкой кисти тёмно-зелёного цвета, добавьте тени в левой части сцены, чтобы приглушить отдельные яркие зелёные тона, которые выделяются.

Увеличьте Жёсткость (Hardness) кисти до 100%. Теперь нарисуйте тени вокруг ботинок. Добавьте тени на земле под ботинками, а также добавьте тени к шнуркам и на язычки ботинок для усиления объёма.

Ниже вы можете увидеть анимацию всего процесса добавления теней До и После. Обратите внимание, как стали выглядеть ботинки после добавления теней.

Шаг 3

Далее, давайте изменим цвет. Чтобы добавить ко всей картине зелёный тон, добавьте корректирующий слой Поиск цвета (Color Lookup). Установите следующие настройки:

  • 3DLUT File: TensionGreen.3DL

Далее, добавьте ещё один корректирующий слой Яркость / Контрастность (Brightnessand Contrast). Установите следующие настройки:

  • Яркость (Brightness):-27
  • Контрастность (Contrast):15

Шаг 4

Старайтесь всегда оставлять работу над бликами ближе к концу урока, когда вы лучше понимаете световую схему. Теперь добавьте новый слой, поменяйте режим наложения для данного слоя на Перекрытие (Overlay). Далее, с помощью большой мягкой круглой кисти белого цвета, прокрасьте правую часть сцены.

Продолжайте добавлять световые блики на ботинки и землю. В данном случае, я добавила блики на язычки ботинок, а также в нижний левый угол сцены.

Обычно ботинки имеют блестящую текстуру кожи. Добавьте немного блеска к ботинкам, добавляя светлые блики на стопы, пальцы ног, а также на детали ботинок. Пусть крючки для шнурков засверкают! Обратите внимание на анимацию ниже, чтобы увидеть эту разницу.

6. Усиливаем Резкость и Добавляем Грязь

Шаг 1

Пришло время для заключительных деталей! Ботинки в лесу не должны выглядеть чистыми. Поэтому, создайте новый слой, чтобы добавить грязь. С помощью кисти Мел (Chalk), добавьте тёмно-коричневую грязь на ботинки.

Шаг 2

Далее, выберите инструмент Рамка (Crop Tool (C). Кадрируйте композицию, чтобы сделать изображение немного меньше. Это поможет привлечь внимание в центр сцены, одновременно усиливая перспективу.

Шаг 3

Для усиления чёткости деталей, мы добавим резкости к нашей сцене с помощью фильтра Цветовой контраст (High Pass Filter). Вначале выделите все слои, в затем объедините их (Ctrl+Alt+Shift+E). Далее, продублируйте объединённый слой (Ctrl+J).

Теперь идём Фильтр - Другое - Цветовой контраст (Filter > Other > High Pass). Установите радиус 3.4 px. Поменяйте режим наложения для слоя с цветовым контрастом на Мягкий свет (Soft Light).

Шаг 4

Продолжаем играть с корректирующими слоями для коррекции цвета и освещения. В качестве заключительного шага, добавьте последний корректирующий слой Поиск цвета (Color Lookup). Установите следующие настройки:

  • 3DLUT File: Fuji F125 Kodak 2395

И на этом всё! Вы можете увидеть итоговый результат на скриншоте ниже!

Мы завершили урок! Давайте отправимся в поход!

Рассмотрите потрясающий набор Природных Экшенов Photoshop от GraphicRiver, чтобы применить естественные, природные цвета к своим работам.
