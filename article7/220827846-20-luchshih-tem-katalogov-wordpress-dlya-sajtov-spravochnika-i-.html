20 лучших тем каталогов WordPress: для-сайтов справочника и ......

Web Design WordPress Themes
Tutorial

Каждая компания ищет возможности продвижения, а сайты каталогов имеют большой потенциал увеличения трафика и монетизации.

В WordPress directory themes есть шаблоны для создания любого веб-сайта: от списков предприятий и местных достопримечательностей до объявлений о работе и сайтов купонов.

Какие признаки отличают лучшие темы WP-каталогов

Помимо обязательной адаптивности и быстрой загрузки, для тем каталога WordPress характерны следующие опции:

  • Простота работы - хорошая тема каталога должна быть удобной для навигации и применения. Качественные темы содержат функции поиска и фильтрации списка.
  • Наличие Google Maps - с его помощью пользователи легко находят все места, указанные в списке, достопримечательности, апартаменты и прочее.
  • Монетизация - простой приём платежей и место для размещения рекламных объявлений.
  • Front-End для пользователей - добавление собственного бизнеса и привлечения внимания к нему.

Это обязательные функции любой темы для создания успешного сайта каталога. Без вариантов.

Популярные темы каталогов WordPress

Темы WordPress directory themes с лучшими параметрами для сайтов местного и мирового значения. Все необходимые функции в достойном оформлении. Качественный сайт каталогов для вас и ваших клиентов.

1. Listify - WordPress Business Directory Theme

Listify - многоцелевая, отзывчивая тема каталога, которая хороша внешне и очень функциональна. Одна из лучших WordPress тем каталогов в этом списке.

Тема с предустановленными стилями, редактируется перетаскиванием в соответствии с задачами вашего бренда для быстрой настройки макетов страниц. Кроме того, функция геолокации Listify выдаст пользователю избранные сведения из списка.

Есть возможности оплаты утверждённого перечня, нового списка с изменениями или заказа с вашего сайта.

Listify - это быстрая загрузка, оптимизация SEO, плагины WooCommerce, FacetWP, WP Job Manager, фильтрация по местоположению или почтовому индексу, а также возможность отправки из внешнего интерфейса.

2. Motors - Directory WP Theme for Vehicle Listings & Classifieds

Тема каталогов WordPress Motors сделана с прицелом на транспортную и лодочную торговлю. Тема оснащена четырьмя потрясающими заготовками макетов. Кроме четырёх макетов, Classified Listing дополнительно имеет возможность регистрации продавцов и дилеров, XML/CSV импорт, профили дилеров с обзорами клиентов и многое другое.

Благодаря расширенному Inventory Manager, можно вводить всю необходимую информацию о транспортном средстве - обзор, фотографии, видео, технические детали, калькулятор кредита и брошюру в формате PDF.

Motors удобно настраивать, благодаря Visual Composer Builder и панели Theme Options, которая поможет уточнить каждый аспект вашего сайта.

3. Superlist - Directory WordPress Theme

Superlist - ещё одна многоцелевая тема каталогов WP с отзывчивым дизайном и множеством опций под капотом.

Superlist поставляется с 12 предустановленными перечнями и возможностью создавать собственные списки в интуитивно понятном интерфейсе Listing Manager. Пользователи могут публиковать свои записи прямо из front-end, а вам будет удобно, благодаря плюсам функциональности темы, формировать пакет оплаты.

Этот шаблон каталогов интегрирован со Stripe и Paypal, что очень упрощает расчёты по заявке. Адаптивный дизайн и разнообразие настроек можно отнести к дополнительным преимуществам.

Наконец, в Superlist можно публиковать купоны, собирать статистику, отслеживая поисковые запросы в фильтрах, управлять счетами, включая Google Maps и прочее.

4. Workscout - Job Board WordPress Theme

Workscout - многофункциональная тема WordPress с бесплатным плагином WP Job Manager для создания неотразимо привлекательного веб-сайта. Пользуйтесь им для построения сайта по трудоустройству!

Благодаря Resume Manager ваши пользователи могут зарегистрироваться и загрузить своё резюме или заполнить предложенную форму на месте. С помощью Application Deadline можно расширить тему, а пользователи получат возможность просматривать вакансии и управлять ими из вашего приложения.

Из других возможностей назовём отзывчивый дизайн, закладки, встроенный Visual Composer Builder и расширенные функции фильтрации поиска.

5. Listable - Friendly Directory WordPress Theme

Listable располагающая к себе тема каталогов WordPress, которая поможет вам создавать, управлять и монетизировать каталоги мирового масштаба.

Благодаря уникальной системе карточек вы можете создавать каталоги событий, мест, видов деятельности и услуг, поставив нужные фильтры.

Listable содержит популярные бесплатные и коммерческие плагины WP Job Manager, WooCommerce, расширяющие возможности продажи услуг по составлению списков и бронированию.

Тема полностью адаптивна и может быть настроена либо с помощью пресетов стиля, либо путем изменения цветов и шрифтов и использования плагина компоновщика страниц, вроде Visual Composer или Beaver Builder.

6. Jobify - WP Job Board Directory Theme

Jobify - это ещё одна тема трудоустройства с готовыми шаблонами страниц, расширенной функцией поиска и фильтрацией по месту для точности.

Тема оснащена плагином WP Job Manager и привязана к Indeed, Facebook, Xing, Woo Testimonials, каждый из которых облегчает и ускоряет создание сайта трудоустройства.

Jobify имеет чистый дизайн, красивые шрифты и все необходимые настройки. С этим шаблоном WordPress ваш совет по поиску работы обязательно выделится из общего ряда.

7. Service Finder - WordPress Business Directory Listing Theme

Service Finder WordPress - продвинутая тема для каталогов бизнеса и услуг. Поражает набором полезных функций, тут и управление профилями и временем бронирования, настройки недоступности, управление персоналом и заказами, создание счетов и платежная система, зоны обслуживания и услуги, поставщики..., и прочее, и прочее.

Тема оснащена плагином WP Job Manager, поддерживает оплату через Stripe и Paypal с возможностью ограничения регистрации по странам и фильтры по месту. У Service Finder два варианта домашней страницы и управление настройками в панели Theme Options.

8. Point Finder - Business Directory WordPress Theme

Point Finder поможет вам создать и запустить сайт каталогов в считанные минуты. Чистая, отзывчивая тема. Примечательны функции настройки поиска и полей темы, загрузка из front-end, система проверок и широкие возможности оплаты.

Тема бизнес-каталога WordPress удобна возможностью управления внешним видом вашего веб-сайта благодаря интеграции с Visual Composer Builder. Среди её полезных опций: social регистрация, геолокация Google Map, шаблоны почты и пользовательские виджеты.

9. Classiads - Classified Ads WordPress Theme

Classiads тема перечней WordPress, предназначенная для создания впечатляющих сайтов бизнес-каталогов. Главные её достоинства - удобство и читабельность, что делает её очень приятной в использовании.

Что касается дизайна, у неё два разных макета и три готовых шаблона с оптимизацией SEO. Функциональность темы обеспечивается настройками полей и возможностью для клиента оформления списка из front-end. Наконец, Classiads оснащена Paypal, MailChimp, and Google Maps.

10. Directory Portal - Directory WordPress Listing Theme

Directory Portal мощная тема WordPress для удобства создания каталогов различной направленности: от местных компаний до событий и достопримечательностей.

Современный, свежий дизайн темы имеет набор из семи видов заголовков, обычный Google Map, 3D Google Map и видео-header.

Поиск и фильтр по геолокации, регистрация и заявки из front-end, проверка гостевых списков, рекламные места, неограниченные пакеты, списки премий - эти функции темы можно назвать среди основных достоинств.

11. MyCity - Geolocation Directory & Events WP Theme

MyCity - великолепная тема каталога WordPress, которая каждого посетителя превратит в клиента, благодаря рекламному виджету, который поощряет делиться интересными местами в социальных сетях, повышая посещаемость.

Наряду со встроенными в неё Stripe и Paypal, тема предлагает уникальные возможности партнёрства с Uber Service для монетизации вашего каталога.

Простая регистрация, поддержка геолокации, обзоры, заказ билетов через Tickera и много других дополнительных опций.

12. Price Compare - Price Comparison WordPress Listing Theme

Если вам нужен очень особенный сайт сравнения цен, не ищите дальше Price Compare. Кроме адаптивного и отзывчивого дизайна, Price Compare позволяет импортировать ссылки партнёров и использовать их сети, вроде Shareasale, ClickBank.

Тема очень удобна по навигации, что крайне важно для сайта каталогов. К услугам ваших посетителей удобные категории, мега-меню и расширенный поиск ajax.

А ещё front-end регистрация магазинов и посетителей, вход через соцсети, встроенный MailChimp и пять вариантов оплаты. Этот шаблон укомплектован профессиональными опциями сравнения цен и прекрасно оформлен.

13. Search & Go - Modern & Smart WP Directory Theme

С помощью Search & Go темы WordPress вы откроете все самые лучшие места своего города. Она очень отзывчива, с ярким, современным дизайном.

Лёгкий вход в систему через социальные сети, расширенные функции поиска и фильтрации. Есть возможность назначать встречи прямо с вашего сайта. А ещё лёгкое управление своими профилями, списками пожеланий и заявок.

В каждый список можно добавить особые условия и удобства, а нужная информация появится в результатах поиска.

14. Javo Spot - Multipurpose Directory WordPress Theme

14. Javo Spot - универсальная тема каталогов WordPress, подходящая любому типу сайтов. Имеются расширенные функции поиска, front-end регистрация и отправка, shortcode обработки списка, WooCommerce, Google Maps и интеграция в социальных сетях.

Этот шаблон бизнес-каталога полностью настраивается благодаря встроенным модулям Visual Composer Builder и Javo. Профессиональный шаблон справится и с задачей создания сайта мирового масштаба, и местных туров, и сайта классификатора, и с любым видом сайта каталогов.

15. Business Finder - Directory Listing WordPress Theme

Business Finder элегантный шаблон темы Wordpress каталогов любого типа и содержания: компаний, событий, магазинов, веб-сайтов и ещё много чего.

Лёгким перетаскиванием элементов в AIT Page Builder, а в нём их более 30, вы быстро построите свой сайт. Мегаменю, sidebar менеджер, расширенные опции поиска и фильтрации, обработка списка, адаптивный дизайн, семь настроек заголовков, просмотр контекста, виды улиц - всё это ваши мощные помощники.

16. Reviews - Products & Services Review WP Theme

Reviews - готовая система обзора продуктов и услуг, где пользователи смогут прочитать о них отзывы, прежде, чем заказывать покупку. Вы можете создать столько обзоров, сколько захотите, а пользователи смогут давать свои отзывы отдельно и оценивать продукт или услугу по вашим критериям.

Тема с опрятным, чистым дизайном, поддержкой Google Rich Snippets, социальных аккаунтов, мощной системой поиска и встроенным MailChimp.

17. Coupon Hut - Coupons & Deals WordPress Theme

CouponHut современная, чистая тема Wordpress, с лёгкой компоновкой страниц, со скидками и купонами, установкой в один клик, множеством виджетов для настройки сайта по своему вкусу. В сочетании с системой оценок и фантастической панелью опций, этот шаблон бизнес-каталога - отличный выбор для сайтов купонов и сделок.

Отличительные функции шаблона: front-page регистрация и поиск с фильтрацией, интеграция членства. Модный, универсальный, детально проработанный дизайн с удобными настройками создания сайта.

18. Medical WordPress - For Doctor Listings & Health Directories

Medical Wordpress тема каталогов специально для врачей, клиник, больниц и медицинских организаций, а также для бизнеса, связанного со здоровьем или красотой.

Главными достоинствами шаблона можно считать фильтрацию по месту и расстоянию, разные виды списков, таймер обратного отсчёта. Плюс календарь бронирования, обзоры рейтингов, встроенный WooCommerce и несколько платёжных шлюзов.

Он снабжён всеми функциями для создания качественного сайта каталога справочников для медицинских работников и связанных с ними организаций.

19. Classiera – Classified Ads Directory WordPress Theme

Classiera мощная тема каталога WordPress с потрясающим дизайном и шрифтами, удобная в работе и лёгкая в настройке стилей через панель администратора.

Располагающее оформление с поиском по фильтрам, настройками категорий списков, WooCommerce, шестью макетами, платными и коммерческими объявлениями выделяют Classiera среди других сайтов объявлений.

20. Alliance - Intranet & Extranet WordPress Directory Theme

Специальная тема Alliance предназначена корпоративным пользователям в локальной сети для составления списка важных событий, уведомлений, календарей и графиков.

Связанная с BuddyPress, позволяет вашим сотрудникам создавать функциональные сообщества, группы и отделы для приятного общения. Предполагается совместное использование файлов, опросы и общий каталог.
