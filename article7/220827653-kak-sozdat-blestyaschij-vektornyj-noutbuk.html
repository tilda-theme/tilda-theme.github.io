Как создать блестящий векторный ноутбук...

Design & Illustration Illustration
Tutorial

В этом уроке я покажу вам, как создать ноутбук в Illustrator, используя базовые инструменты, такие как инструмент Rectangle Tool, Ellipse Tool, Pen Tool и другие. Мы построим общую форму, клавиатуру, монитор для ноутбука и многое другое.

Шаг 1

Откройте новый документ, выберите инструмент Rectangle (M) и создайте прямоугольник размером 504 px на 377,5 px и заполните его черным цветом.

Шаг 2

Скопируйте только что созданный прямоугольник с помощью Command + C, за которым следует Command + F. Возьмите инструмент Selection (V) и немного уменьшите размер копии в нижней части и заполните Linear Gradient. Для этого откройте панель Gradient (Command + F9) и выберите тип: Linear. Вам нужны два Gradient Sliders. Для первого слева направо выберите цвет: R = 20, G = 27, B = 34, а для второго градиентного слайдера - R = 51, G = 51, B = 51.

Шаг 3

Скопируйте прямоугольник, заполненный линейным градиентом, с помощью Command + C, а затем Command + F. С помощью Pen tool (P) создайте фигуру поверх копии, как показано. Теперь выберите копию и созданную вами фигуру, откройте панель Pathfinder (Shift + Command + F9), нажмите кнопку Minus Front и заполните Linear Gradient. Вам нужно два Gradient Sliders. Для обоих из них выберите белый цвет, так как для первого слайдера нужно уменьшить непрозрачность до 0.%.

Шаг 4

Скопируйте фигуру, которую вы только что заполнили линейным градиентом в шаге 2, с помощью Command + C, а затем Command + F. Теперь переместите прямоугольник над всеми остальными, перейдя в Object> Arrange> Bring To Front (Shift + Command + Right Bracket key).

Измените цвет заливки на черный. Возьмите инструмент выделения (V), нажмите на нижнюю или верхнюю часть в середине прямоугольника, удерживайте и нажмите клавишу Alt и немного сдвиньте его внутрь. Таким образом, вы уменьшите верхнюю и нижнюю части равномерно. Повторите то же действие для боковой части, посмотрите на картинку ниже, чтобы получить лучшую идею.

Шаг 5

Скопируйте только что созданный прямоугольник и заполните его черным цветом с помощью Command + C, а затем Command + F. Возьмите инструмент Direct Selection Tool (A) и удалите верхнюю и нижнюю левую точку прямоугольника, нажав на одну точку, затем нажмите клавишу Delete. Повторите то же действие для другой левой точки, выключите цвет заливки и примените цвет обводки R = 189, G = 204, B = 212 с весом 1px.

Шаг 6

Теперь выберите прямоугольник из шага 4, который вы заполнили черным цветом, затем перейдите в Object> Path> Offset Path в новом окне открытия. Теперь установите смещение - 2px и нажмите OK.

Шаг 7

Заполните прямоугольник, созданный на шаге 6, с помощью линейного градиента после применения смещения. Для этого откройте панель Gradient (Command + F9). Вам нужно два Gradient Sliders. Для первого слева на право выберите цвет: R = 47, G = 50 и B = 52 и второй выберите черный цвет.

Шаг 8

Скопируйте последний созданный прямоугольник, который вы заполнили линейным градиентом на шаге 7, с помощью Command + C, за которым следует Command + F. Создайте фигуру в такой позиции, как показано ниже. Откройте панельPathfinder и нажмите на Minus Front. Заполните созданную форму с помощью Linear Gradients. Для обоих Gradient Sliders выберите белый цвет, а затем уменьшите Opacity до 0% для первого Gradient Slider.

Шаг 9

Теперь давайте создадим веб-камера на ноутбуке. Выберите инструмент Ellipse Tool (L), создайте круг в положении размером 16px/16px, как показано на рисунке ниже, и заполните его черным цветом.

Шаг 10

Когда круг все еще выбран, перейдите в Object> Path> Offset Path, установите Offset на -2px, нажмите OK и заполните цветом: R = 189, G = 204, B = 212.

Шаг 11

Теперь с последним кругом вы применили Offset to still selected, снова перейдите Object> Path> Offset Path и в новом окне открытия установите Offset to -2px, нажмите OK и заполните черным цветом.

Шаг 12

Повторите то же действие для последнего круга и заполните его белым цветом. Снова примените смещение к кругу, заполненному белым цветом, и измените цвет заливки на черный.

Шаг 13

Выберите инструмент Ellipse (L). Создайте круг с размерами 8px на 8px в положение, которое покано. Когда круг, заполненный черным цветом, по-прежнему выбран, перейдите в Object> Path> Offset Path, а в новом открытом окне установите Offset на 0,5px. Теперь заполните новый созданный круг линейным градиентом с цветами от белого до черного.

Шаг 14

Сгруппируйте последние два созданных круга, скопируйте их три раза и расположите их в четырех углах монитора ноутбука.

Шаг 15

Выберите Rounded Rectangle Tool, создайте эллипс размером 6px на 32px и угловой радиус 20px. Заполните линейный градиент цветами от черного до белого и установите эллипс на левой стороне монитора.

Шаг 16

Скопируйте эллипс с помощью Command + C, а затем Command + F, переместите его в правую сторону монитора.

Шаг 17

Создайте круги в положениях, показанных ниже.

Шаг 18

Возьмите Pen Tool (P) и создайте четыре угла формы монитора, показанные ниже. Выберите четыре фигуры и примените белый цвет обводки и 1px Weight.

Шаг 19

Снова используйте Pen Tool (P), чтобы создать линию с конца одной из фигур, сделанных на предыдущем шаге, до конца другой фигуры на двух противоположных фигурах. Посмотрите на изображение ниже для пояснения. Теперь примените обводку белого цвета и вес 0,25px.

Шаг 20

Теперь выберите все формы монитора и сгруппируйте их. Для этого перейдите в Object> Group (Command + G).

Шаг 21

Давайте создадим другую часть нашего ноутбука. Все формы, которые мы создадим, будут находиться под группой мониторинга. Выберите инструмент Rectangle (M) и создайте прямоугольник с разрешением 458px на 10px, установите штрих в черный цвет и поместите его в нижней части монитора.

Шаг 22

Скопируйте прямоугольник с помощью Command + C, за которым следует Command + F, выберите Selection Tool (V), щелкните правой или левой средней точкой, нажмите и удерживайте клавишу Alt и перемещайте внутрь. Таким образом, вы равномерно уменьшите обе стороны прямоугольника, затем заполните его линейным градиентом с цветами от черного до серого.

Шаг 23

Повторяя те же действия, создайте еще два прямоугольника как на левой, так и на правой сторонах, когда вы оставите расстояние в несколько пикселей между ними.

Шаг 24

Возьмите инструмент Rectangle (M) и создайте прямоугольник размером 575 px на 154 px и расположите его в нижней части монитора.

Шаг 25

Теперь, когда последний прямоугольник, который вы создали, все еще выбран, перейдите в Effect> Distort & Transform> Free Distort и сдвиньте верхние две точки внутрь, как показано. После этого примените Object> Expand Appearance.

Шаг 26

Теперь возьмите инструмент Add Anchor Point Tool (+), добавьте две точки с правой стороны, исправьте их, как показано, и заполните их линейным градиентом. Что касается первого Gradient Slider, примените цвет: R = 202, G = 206, B = 208, для второго R = 106, G = 109, B = 112 и установите угол 90.

Шаг 27

Возьмите инструмент Rectangle Tool (M) и создайте прямоугольник размером 575 px на 20 px и поместите его в нижнюю часть ранее созданного прямоугольника, как показано.

Шаг 28

Возьмите инструмент Add Anchor Point Tool (+), добавьте две точки в нижней части прямоугольника и сдвиньте их на несколько пикселей вниз с помощью инструмента Direct Selection Tool (A). Теперь заполните их линейным градиентом, так как для первого Gradient Slider выберите цвет R = 189, G = 204, B = 212, а для второго - белый.

Шаг 29

Создайте прямоугольник 570 px на 74 px с помощью инструмента Rectangle (M), заполните его цветом R = 18, G = 25, B = 33 и поместите его, как показано.

Шаг 30

Скопируйте прямоугольник с шага 25, который вы использовали Free Distort для использования (Command + C, а затем Command + F) и перейдите в Object> Arrange> Bring to Front (Shift + Command + Right Bracket key). Теперь выберите копию и прямоугольник с предыдущего шага и перейдите в Object> Clipping Mask> Make (Command + 7).

Шаг 31

Выберите прямоугольник, примененные обтравочную маску, чтобы выбрать инструмент точка привязки добавить (+), добавить четыре точки, как показано, затем сдвиньте две средние точки немного вниз.

Шаг 32

Скопируйте форму, на которую вы применили Clipping Mask, с помощью Command + C, а затем Command + F. Используйте инструмент Direct Selection Tool (A), чтобы удалить нижние две точки.

Шаг 33

Теперь отключите цвет заливки и примените цвет обводки с Weight 5. Неважно, какой цвет вы выбираете, потому что мы собираемся заполнить фигуру линейным градиентом. С помощью фигуры, которую вы применили, цвет Stroke до сих пор выбран, перейдите в Object> Path> Outline Stroke и заполните Linear Gradient.

Для этого откройте панель Gradient (Command + F9) и выберите Linear для Type. Вам нужно три Gradient Sliders вместе. Для первого используйте белый цвет с 68% местоположением, для второго - темно-серый цвет с 72% местоположением, а для третьего - белый цвет с 76% местоположением.

Шаг 34

Используйте инструмент Rectangle (M) для создания прямоугольника размером 575 px на 6 px. Выберите черный цвет и положение, как показано.

Шаг 35

Теперь используйте инструмент Add Anchor Point Tool (+), чтобы добавить две точки в нижней части прямоугольника. Немного сдвиньте нижнюю левую точку, добавленную ранее, посмотрите на рисунок ниже.

Шаг 36

Добавьте световые сигналы и гнезда в течение последней созданной формы, чтобы сделать ее более реалистичной.

Шаг 37

Создайте прямоугольник с помощью инструмента Rectangle Tool (M) в 136 px на 43 px в показанном положении, затем заполните его серым цветом.

Шаг 38

Когда последний созданный прямоугольник все еще выбран, перейдите в Object> Path> Offset Path и в новом окне установите Offset в -1px и нажмите OK. Теперь измените цвет на более легкий.

Шаг 39

Создайте еще один прямоугольник, используя инструмент Rectangle Tool (M) с размерами 133 px на 11 px. Выберите черный цвет и поместите его, как показано ниже.

Шаг 40

Создайте еще два меньших круга над последним черным прямоугольником и заполните их R = 202, G = 206, B = 208.

Шаг 41

Теперь нужно создать кнопки для ноутбука. Выбрать инструмент прямоугольник (M) и создайте прямоугольник с размерами 21,5 px, 19,5 px.

Шаг 42

Скопируйте прямоугольник с помощью Command + C, за которым следует Command + F, увеличьте размер копии с помощью инструмента выделения. Теперь используйте Pen Tool (P), чтобы создать линии на маленьком прямоугольнике в углу большого прямоугольника. Затем выберите все фигуры, откройте панель Pathfinder и нажмите кнопку Divide.

Шаг 43

Теперь заполните вновь созданные фигуры. Левая форма заполняется линейным градиентом с цветами от серого до черного. Заполните нижнюю форму тем же градиентом, а остальные фигуры залиты черным цветом.

Шаг 44

Выделите все формы кнопки и перейдите в Effect> Distort & Transform> Transform, а в новом открывшемся окне установите значение «12», Horizontal до 28 пикселей и нажмите «ОК».

Шаг 45

Когда фигуры все еще выбраны, снова перейдите в Effect> Distort & Transform> Transform, а в новом окне открытия установите значение 1, Horizontal значение до 40 пикселей, Вертикальное значение -27 px и нажмите OK.

Шаг 46

При повторном выборе фигур перейдите в Effect> Distort & Transform> Transform и в новом открывшемся окне установите Копии на 2, Вертикально на -54 px и нажмите OK.

Шаг 47

С выбранными кнопками перейдите в Object > Expand Appearance и исправьте кнопки с помощью Direct Selection tool (A), как показано.

Шаг 48

Добавьте необходимый знак на каждую кнопку. Выберите значки и кнопки, затем сгруппируйте их, применив Object> Group (Command + G).

Шаг 49

С выбранными кнопками откройте панель символов (Shift + Command + F11) и сдвиньте кнопки на панели Symbol.

Шаг 50

Теперь вы можете преобразовать кнопки так, как вы хотите, и на этом остановиться.

Шаг 51

Я покажу вам другой способ применения кнопок. с кнопками Symbol все еще выбраны, перейдите в Effect> Distort & Transform> Free Distort, а в новом открытом окне измените кнопки, как показано на рисунке, затем нажмите OK.

Шаг 52

После того, как вы изменили кнопки, снова откройте панель Symbol (Shift + Command + F11) и сдвиньте их еще раз на панели Symbol.

Шаг 53

Теперь используйте Pen Tool (P), чтобы создать форму, на которую вы будете накладывать кнопки, посмотрите на изображение ниже для справки.

Шаг 54

Выберите последнюю форму, на которую вы будете накладывать кнопки, и перейдите в Effect> 3D> Extrude & Bevel, установите для всех параметров значение 0, затем нажмите кнопку «Art Art» и примените «символ 2», который вы создали.

Шаг 55

Используйте Pen Tool (P) для создания фигур с обеих сторон рядом с кнопками. Заполните их линейным градиентом. Что касается обоих градиентных слайдеров, используйте белый для обоих, а затем уменьшите непрозрачность для второго ползунка градиента до 0%.
