Создаём-Сюрреалистическую Фотоманипуляцию "Расступившееся Море"...

Design & Illustration Photo Effects
Tutorial

Воду по своей природе сложно передать в композиции. В этом уроке вы узнаете, как сочетать продвинутые техники работы с маской, режимами наложения, корректирующими слоями, как правильно использовать фильтры для разделения моря, а также как создать сюрреалистическую фотоманипуляцию. Давайте приступим!

Шаг 1

Установите цвет Заднего фона (Background) #e7f3fe. Создайте новый документ 27cm x 18cm, установите Разрешение (Resolution) 300dpi, Цветовой режим (Color Mode) RGB, Содержимое Заднего фона (Background Contents) на Цвет Заднего фон (Background Color). Это будет наш рабочий документ.

Шаг 2

Откройте первое исходное изображение с пейзажем, а затем идём Изображение - Поворот изображения - Отразить по горизонтали (Image > Image Rotation > Flip Horizontal). Переместите данное изображение на наш рабочий документ, а затем примените трансформацию таким образом, чтобы закрыть верхнюю половину холста. Назовите этот слой с пейзажем Горизонт (Horizon).

Шаг 3

Нажмите клавишу "D", чтобы установить цвет переднего плана на чёрный цвет. Выберите инструмент Градиент (Gradient Tool (G), а затем в панели управления данного инструмента, установите тип градиента От переднего плана к прозрачному (Foreground to Transparent), а также Линейный (Linear) градиент. Теперь, удерживая клавишу (Shift), потяните градиент сверху в направлении красной стрелки, чтобы совместить верхнюю часть изображения с голубым задним фоном, как показано на скриншоте ниже. Примечание переводчика: не забудьте добавить слой-маску к слою с пейзажем, а затем на маске слоя, скройте верхнюю часть пейзажа.

Шаг 4

Убедитесь, чтобы слой был активным, а не маска, а затем в нижней части панели слоёв, нажмите значок Создать новый корректирующий слой или слой заливку(Create new fill or adjustment layer) и в появившемсяменю, выберите опцию Уровни (Levels). В появившемся окне настроек уровней, нажмите на значок отсечения по слою, чтобы коррекция воздействовала только на нижний слой. Установите настройки, которые указаны на скриншоте ниже, чтобы усилить контраст. Примечание переводчика: чтобы слой был активным, а не маска, просто щёлкните по миниатюре слоя.

Шаг 5

Для нашей сцены необходимо захватывающее небо. Откройте второе исходное изображение с пейзажем. С помощью инструмента Лассо (Lasso Tool (L), создайте выделение вокруг птицы. Нажмите клавиши (Shift+F5), чтобы появилось окно заливки. Из выпадающего меню, выберите опцию С учётом содержимого (Content-Aware).

Переместите данное изображение с пейзажем на наш рабочий документ, расположив поверх всех остальных слоёв. Уменьшите изображение по вертикали, как показано на скриншоте ниже. Назовите этот слой Небо (Sky), а затем примените ту же самую технику, которая описана в Шаге 3, чтобы совместить нижний жёсткий край с горизонтом.

Шаг 6

К слою Небо (Sky), добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски, чтобы усилить контраст.

Шаг 7

Добавьте ещё один корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски. Установите следующие настройки, которые указаны на скриншоте ниже. Цвет и тон обоих слоёв теперь должны идеально сочетаться.

Шаг 8

У Photoshop есть много инструментов, которые помогают нам отделить изображения от заднего фона; большинство рассчитано на работу с хорошим контрастом между объектом и задним фоном. В этом уроке мы рассмотрим различные техники выделения объектов.

Для нашего первого изображения, нет быстрого решения для создания точного выделения, поэтому мы используем канал или маску плотности. Откройте третье исходное изображение с пейзажем, а затем идём Изображение- Вычисления (Image > Calculations). В данном случае, оба синих канала установлены в режиме наложения Затемнение основы (Color Burn), чтобы создать наилучший контраст в качестве исходной точки для нашей маски. Вам также необходимо выбрать Новый канал (New channel) из выпадающего меню Результат (Result).

Шаг 9

Перейдите во вкладку Каналы (Channels) и вы увидите новый канал Альфа 1 (Alpha 1) в нижней части палитры, при этом видимость других каналов будет отключена. Чтобы посмотреть действие канала на изображении, включите видимость RGB канала. Для облегчения просмотра, дважды щёлкните по миниатюре канала, а затем щёлкните по цветному квадратику, чтобы поменять цвет (красный цвет стоит по умолчанию), а также поменяйте значение Непрозрачности (Opacity) на подходящее.

Следующий этап - это созданий чёткого силуэта, поэтому, убедитесь, чтобы ваш новый канал был активным. Выберите инструмент Кисть (Brush (B) Жёсткостью (Hardness) примерно 50%. Удалите нижнюю левую область, как показано на скриншоте ниже. Не забудьте включать / выключать видимость RGB канала во время работы.

Шаг 10

Выберите инструмент Осветлитель (Dodge Tool (O), установите следующие настройки: Диапазон (Range) Подсветка (Highlights), Экспозиция (Exposure) 65%, используйте среднего размера кисть с мягкими краями. Теперь прокрасьте кистью вдоль внешнего края деревьев.

Шаг 11

С помощью кисти, чередуя цвет кисти чёрный и белый, аккуратно проведите коррекцию маски канала.

Шаг 12

По умолчанию, белый цвет представляет выделенные участки канала, поэтому нажмите клавиши (Ctrl+I) для инверсии канала на противоположный цвет. Инверсия канала также может выявить небольшие дефекты, которые могут быть исправлены с помощью Кисти (Brush Tool (B).

Шаг 13

Удерживая клавишу (Ctrl), щёлкните по миниатюре канала, чтобы загрузить активное выделение. Переключитесь на канал RGB, а затем перейдите обратно во вкладку Слои (Layers). Теперь нажмите клавиши (Ctrl+J), чтобы скопировать выделение на новый слой, а затем отключите видимость основного слоя, чтобы увидеть результат.

Шаг 14

Переместите выделенное изображение на наш рабочий документ, назовите этот слой Средний план 1 (Midground 1). Далее, идём Трансформирование - Отразить по горизонтали (Transform > Flip Horizontal) для масштабирования. Расположите выделенное изображение пейзажа в левой части сцены. Чтобы удалить возможное свечение вдоль краёв, идём Слой – Обработка края - Устранить кайму (Layer > Matting > Defringe) и в появившемся окне, установите значение 1рх.

Шаг 15

Далее мы удалим нежелательные коричневые тона. Находясь на слое Средний план 1 (Midground 1), идём Изображение - Коррекция - Заменить цвет (Image > Adjustments > Replace Color). Установите значение Разброса (Fuzziness) на 88, чтобы увеличить диапазон выделения, а затем замените Цветовой тон (Hue) на +68. Теперь с помощью пипетки, щёлкните по самому тёмно-коричневому участку в окне предварительного просмотра. Для удаления схожих оттенков, используйте пипетку с "+".

Шаг 16

Для выделения объекта на четвёртом изображении пейзажа, мы также используем маску канала. Ещё раз откройте окно Вычисления (Calculations), только на этот раз для обоих синих каналов установите режим наложения Линейный затемнитель (Linear Burn).

Шаг 17

На этом канале отсутствует контраст, как на предыдущем изображении. Поэтому, вместо применения техники осветления поверх всего изображения, нажмите клавиши (Ctrl+L), чтобы открыть окно коррекции Уровней (Levels). Теперь с помощью белой пипетки для определения точки белого, щёлкните несколько раз по любым участкам серого тона, чтобы сделать их белыми.

Шаг 18

Повторите ту же самую технику коррекции и создания инвертированной маски канала.

Шаг 19

Теперь скопируйте выделенную область на новый слой, предварительно загрузив маску канала.

Шаг 20

Переместите выделенное изображение на наш рабочий документ. Отразите данное изображение по горизонтали. расположив в правой части сцены. Скройте любое свечение вокруг краёв изображения, а затем назовите этот слой Средний план 2 (Midground 2).

Шаг 21

Повторите действие Замена цвета (Replace Color) на данном изображении, чтобы преобразовать коричневые тона в зелёные.

Шаг 22

Далее мы заполним пустое пространство на переднем плане небольшим количеством песка. С помощью инструмента Лассо (Lasso Tool (L), выделите произвольный фрагмент песка на изображении слоя Средний план 2 (Midground 2). Нажмите клавиши (Shift+F6), чтобы открыть окно Растушёвка (Feather), установите значение растушёвки 5px. Не снимая активного выделения, нажмите клавиши (Ctrl+J), чтобы скопировать выделенный фрагмент песка на новый слой. Сместите скопированный фрагмент песка влево, а затем увеличьте масштаб, как показано на скриншоте ниже.

Шаг 23

Продублируйте слой с фрагментом песка несколько раз, а затем примените масштабирование, закрыв оставшиеся пустоты.

Шаг 24

Удерживая клавишу (Shift), выделите все слои с фрагментами песка, а затем нажмите клавиши (Ctrl+E), чтобы объединить слои.

Шаг 25

Добавьте слой-маску к объединённому слою с песком, далее, нажмите клавиши (Сtrl+I) для инверсии маски в чёрный цвет. Теперь с помощью белой кисти с различной степенью непрозрачности, восстановите текстуру песка. Чтобы вам было понятно, где обрабатывать кистью, ниже я привёл скриншот коррекции маски.

Шаг 26

Выделите объединённый слой с песком и слой Средний план 2 (Midground 2), а затем ещё раз нажмите клавиши (Ctrl+E), чтобы объединить эти слои.

Шаг 27

В следующих шагах мы изменим цвет и контрастность обоих слоёв со средним планом. Вначале добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски к слою Средний план 1 (Midground 1). Установите следующие настройки, которые указаны на скриншоте ниже.

Шаг 28

Далее, добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски к тому же самому слою со средним планом 1. Поменяйте настройки для Средних тонов (Midtones) и Света (Highlights).

Шаг 29

Чтобы применить корректирующий слой Цветовой Баланс (Color Balance) только к участку песка под листвой, для этого проведите инверсию маски данного корректирующего слоя, а затем с помощью белой кисти, прокрасьте участки песка под листвой, восстановив цветовую коррекцию.

Шаг 30

Теперь добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски к слою Средний план 2 (Midground 2). Установите следующие настройки, которые указаны на скриншоте ниже.

Шаг 31

В качестве заключительной коррекции, добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски к тому же самому слою со средним планом 2.

Шаг 32

Далее, мы добавим песок на дальний план на горизонте. Перейдите на слой Средний план 2 (Midground 2). Выберите инструмент Прямоугольная область (Rectangular Marquee Tool (M). Создайте выделение, как показано на скриншоте ниже. Установите радиус Растушёвки (Feather) выделенной области на 2px, а затем скопируйте выделение (Ctrl+C).

Теперь перейдите на слой с корректирующим слоем Уровни (Levels), который расположен поверх слоя Горизонт (Horizon), а затем вклейте скопированное изображение песка на новый слой (Ctrl+V). Примените масштабирование, расположив участок песка таким образом. чтобы закрыть полностью поверхность воды. Назовите слой с этим фрагментом песка Песок вдали (Distant sand).

Шаг 33

Откройте исходное изображение с туристом. Данный снимок сделан на белом фоне, поэтому, выделить объект будет намного проще.

Выберите инструмент Волшебная палочка (Magic Wand Tool (W), в панели управления данного инструмента, выберите режим Добавить к выделенной области (Add to selection), а также установите значение Допуска (Tolerance) на 7, не забудьте поставить галочку в окошках Сглаживание (Anti-alias) и Смежные пиксели (Contiguous). Теперь щёлкните по белому заднему фону, включая участок между рубашкой и рукой.

Шаг 34

Нажмите клавиши (Shift+Ctrl+I) для инверсии выделенной области, в затем нажмите на значок Уточнить край (Refine Edge) в верхней панели настроек. В появившемся окне выберите режим просмотра На чёрном (On Black (B) из выпадающего меню, это поможет лучше увидеть участки удаления белого фона. Теперь поменяйте Умный радиус (Smart Radius) на 0,5рх; поставьте галочку в окошке Очистить цвета (Decontaminate Colors), укажите Количество (Amount) 100%, а также в выпадающем меню Вывод в (Output to) выберите опцию Новый слой с маской (New Layer with Layer Mask).

Шаг 35

Далее вы увидите дубликат слоя с маской, при этом видимость оригинального слоя будет отключена. На этом этапе, возможно потребуется небольшая коррекция маски. Во-первых, создайте новый слой между оригинальным слоем с туристом и дубликатом слоя с маской. Залейте этот слой чёрной заливкой, чтобы увидеть оставшиеся фрагменты белого фона. Теперь увеличьте масштаб, а затем с помощью кисти диаметром 3рх, и Жёсткостью (Hardness) примерно 50%, проведите коррекцию маски по мере необходимости.

Шаг 36

Если вы довольны качеством выделения, перетащите иконку маски на значок корзины в нижней части палитры слоёв и в появившемся окне, нажмите кнопку Применить (Apply).

Переместите выделенное изображение мужчины на наш рабочий документ, расположив поверх всех остальных слоёв. Назовите этот слой Мужчина (Figure). Примените функцию Устранить кайму (Defringe) на 1px, а затем уменьшите масштаб изображения мужчины, расположив его в центре сцены на песке.

Шаг 37

Откройте исходное изображение с береговой линией, а затем с помощью инструмента Лассо (Lasso (L), создайте примерное выделение вокруг береговой линии, исключая мячик. Скопируйте / вклейте выделенный фрагмент берега ниже слоя Мужчина (Figure). Примените трансформацию, расположив кромку берега в левой части сцены. Теперь удерживая клавишу (Ctrl), потяните за угловые точки, чтобы придать фрагменту берега перспективу пляжа.

Шаг 38

Продублируйте слой с фрагментом берега, отразите по горизонтали дубликат слоя, расположив его в правой части сцены. С помощью инструмента Лассо (Lasso (L), выделите небольшие участки кромки воды с морской пеной, установите радиус Растушёвки (Feather) на 3px. Скопируйте выделение на несколько новых слоёв. Теперь перераспределите созданные кромки с морской пеной так, чтобы береговая линия не повторяла в точности береговую линию слева. Мои дополнительные слои с кромкой воды показаны на скриншоте ниже.

Шаг 39

Объедините слои с кромками воды вместе со слоями с береговыми линиями с двух сторон в один слой, назовите этот слой Основа воды (Base water). К объединённому слою Основа воды (Base water), добавьте слой-маску и с помощью чёрной кисти с различной степенью непрозрачности, прокрасьте жёсткие края, чтобы создать плавное совмещение.

Шаг 40

Далее, к объединённому слою Основа воды (Base water), добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски для осветления. Установите следующие настройки, которые указаны на скриншоте ниже.

Шаг 41

Далее мы приступаем к созданию возвышающийся стены из воды с обеих сторон. Откройте исходное изображение с волной, переместите данное изображение на наш рабочий документ, расположив ниже слоя Мужчина (Figure). Примените масштабирование, расположив волну в левой части сцены. Создайте дубликат слоя, отразите по горизонтали, расположив дубликат слоя в правой части сцены. Объедините оба слоя, назовите объединённый слой Возвышающая Волна 1 (Rising water 1).

Шаг 42

Добавьте слой-маску к объединённому слою с волной, а затем аккуратно смягчите верхние и нижние края.

Шаг 43

Чтобы дополнительно совместить волны, добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски, установите значение Средних тонов (midpoint) на 1.34.

Шаг 44

Шаг 90

После применения инструмента Уточнить край (Refine Edge), примените слой-маску. Переместите выделенное изображение черепахи на наш рабочий документ, расположив ниже слоя с группой Объекты на Земле (LAND OBJECTS). Примените трансформацию, отразите по горизонтали, расположив черепаху в правой части нашей сцены. Назовите слой с черепахой Черепаха (Turtle).

Уменьшите Непрозрачность (Opacity) слоя с черепахой до 70%, чтобы создать ощущение удалённости, а затем добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски.

Шаг 91

Теперь добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски к слою Черепаха (Turtle), чтобы получше совместить черепаху с подводной сценой.

Шаг 92

Откройте второе исходное изображение с акулой. Используйте ту технику выделения, с которой вы предпочитаете работать - я использовал значение растушёвки контура выделенной области 2рх. Переместите выделенное изображение акулы на наш рабочий документ, расположив ниже слоя с группой Объекты на Земле (LAND OBJECTS). Назовите этот слой с акулой Акула 2 (Shark 2). Примените трансформацию, расположив акулу в левой части нашей сцены, а затем добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски.

Шаг 93

Добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски к тому же самому слою с акулой. Усильте значение Синих (Blue) для Средних тонов (Midtones) до +20.

Шаг 94

Используя техники выделения, о которых вы узнали, вырежьте стайку рыбок на новый слой. Переместите выделенное изображение стайки рыбок на наш рабочий документ, расположив ниже слоя с группой Объекты на Земле (LAND OBJECTS). Назовите этот слой Рыбки (Fish). Примените трансформацию, расположив стайку рыбок над второй акулой в левой части нашей сцены. Уменьшите Непрозрачность (Opacity) слоя с рыбками до 45%. Теперь добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски к слою с рыбками, чтобы усилить контраст.

Шаг 95

В качестве заключительной коррекции, добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски к слою Рыбки (Fish).

Шаг 96

Теперь сгруппируйте все слои с подводными обитателями, включая обтравочные корректирующие слои в одну группу. Назовите эту группу Подводные обитатели (UNDERWATER CREATURES).

Шаг 97

Откройте исходное изображение с океаном, а затем с помощью инструмента Прямоугольная область (Rectangular Marquee Tool), выделите водную поверхность. Скопируйте /вклейте выделенную область на новый слой. Переместите выделенное изображение на наш рабочий документ, расположив поверх всех остальных слоёв. Поверните данное изображение на 90 градусов по часовой стрелке.

Шаг 98

Теперь деформируйте слой в соответствии с изгибом контура волны слева.

Шаг 99

Продублируйте слой, отразите по горизонтали, расположив на противоположной стороне, а затем объедините оба слоя. Нажмите клавиши (Ctrl+A), чтобы выделить полностью холста. а затем нажмите клавиши (Ctrl+C), чтобы скопировать выделенную область.

Шаг 100

Нажмите клавиши (Ctrl+N), чтобы создать новый документ. В появившемся окне, в окне Набор (preset), укажите Буфер обмена (Clipboard), а также установите Цветовой режим (Color Mode) В градациях серого (Greyscale) и Содержимое фона (Background Contents) белый. Теперь вклейте скопированное выделение (Ctrl+V), а затем выполните сведение.

Далее, идём Фильтр - Размытие - размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите Радиус (Radius) размытия на 5px. Теперь нажмите клавиши (Ctrl+L), чтобы открыть окно уровней. Установите следующие настройки, которые указаны на скриншоте ниже. Мы используем данный документ для искажения подводных обитателей, поэтому, сохраните этот документ как "Displacement.psd" в удобном для вас месте.

Шаг 101

Нам больше не пригодится объединённый слой с Шага 99, поэтому, удалите его. Далее, продублируйте слой с группой Подводные обитатели (UNDERWATER CREATURES), а затем нажмите клавиши (Ctrl+E), чтобы выполнить сведение дубликата слоя. Назовите сведённый слой Эффект Смещения Подводных обитателей (Underwater creatures displaced), а затем отключите видимость оригинального слоя с группой. Чтобы правильно применить фильтр в следующем шаге, выделите полностью холст (Ctrl+A), а затем идём Изображение- Рамка (Image > Crop).

Шаг 102

Убедитесь, что вы находитесь на слое Эффект Смещения Подводных обитателей (Underwater creatures displaced), а затем идём Фильтр - Искажение - Смещение (Filter > Distort > Displace). Установите оба значения Масштаба (Scale) на 12, а также поставьте галочки, чтобы выбрать параметры Растянуть (Stretch To Fit) и Повторить граничные пиксели (Repeat Edge Pixels), а затем нажмите кнопку "OK", далее, появится окно навигации, чтобы вы указали место, где сохранён ваш документ "Displacement.psd", выберите его .

Фильтр Смещение (Displace) действует так же, как карта рельефа в в 3D приложении; он использует информацию о полутонах серого для нажима и искажения пикселей. Фильтр проделал отличную работу, имитируя эффект преломления в подводной среде, но мы ещё не закончили.

Шаг 103

Поменяйте режим наложения для слоя Эффект Смещения Подводных обитателей (Underwater creatures displaced) на Перекрытие (Overlay), а затем добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски. Установите настройки, как показано на скриншоте ниже.

Шаг 104

Нам нужно показать задний фон, который будет виден сквозь толщу волны. Продублируйте слой с группой Передний / Задний план (BACKGROUND/FOREGROUND), выполните сведение дубликат слоя с группой (Ctrl+E). Назовите объединённый слой Размытый Задний Фон (Background blur), расположите его ниже слоя Эффект Смещения Подводных обитателей (Underwater creatures displaced).

Шаг 105

К слою Размытый Задний Фон (Background blur), примените фильтр Размытие по Гауссу (Gaussian Blur), установите радиус размытия 7рх, а затем уменьшите Непрозрачность (Opacity) до 16%. Далее, добавьте инвертированную маску и с помощью большой мягкой кисти белого цвета, прокрасьте участки заднего фона, расположенные за толщей воды. В заключение, удерживая клавишу (Ctrl), щёлкните по миниатюре слоя Эффект Смещения Подводных обитателей (Underwater creatures displaced), чтобы загрузить выделение., а затем залейте выделение чёрным цветом на маске слоя Размытый Задний Фон (Background blur).

Шаг 106

В следующих шагах мы добавим ещё деталей к воде. Вначале с помощью инструмента Лассо (Lasso Tool (L), создайте произвольное выделение вокруг участка волны на изображение серфинга. Установите радиус Растушёвки (Feather) выделенной области на 5px, а затем скопируйте / вклейте выделение на новый слой поверх всех слоёв внутри группы Вода (WATER).

Назовите этот слой Пена 1 (Foam 1), а затем примените Трансформацию (Transform) и Деформацию (warp) к выделенному фрагменту волны. Поменяйте режим наложения для данного слоя на Осветление (Screen), а затем дважды щёлкните по данному слою, чтобы открыть окно Параметров наложения (Blending Options). В нижней части Наложение, если (Blend If), установите Градация серого (Grey), а затем удерживая клавишу (Alt), сместите (разделите) половинку верхнего бегунка вправо до значения 235. Теперь в нижней части Наложение, если (Blend If), установите Зелёный (Green), а затем удерживая клавишу (Alt), сместите ту же самую половину бегунка вправо до значения 140. В заключение, добавьте слой-маску и с помощью чёрной кисти, скройте жёсткие края. Примечание переводчика: с помощью трансформации и деформации, придайте фрагменту волны изгиб контура волны.

Шаг 107

Продублируйте слой Пена 1 (Foam 1) несколько раз, а затем трансформируйте дубликаты слоёв, расположив их на обоих волнах. Ещё раз примените команду Деформации (Warp), чтобы также изменить маски дубликаты слоёв. Переименуйте дубликаты слоёв соответственно.

Шаг 108

Откройте данное изображение с волной, а затем идём Выделение - Цветовой Диапазон (Select > Color Range), щёлкните по самой белой области изображения, установите Разброс (Fuzziness) на 99, а Диапазон (Range) на 33%. Скопируйте выделение (Ctrl+C).

Шаг 109

Вклейте скопированное выделение на новый слой поверх слоя с группой Объекты земли (LAND OBJECTS). Назовите этот слой Гребень волны 1 (Spray 1). Примените трансформацию, а также расположите изображение, как показано на скриншоте ниже, а затем поменяйте режим наложения на Осветление (Screen). Теперь добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски к слою с гребнем волны для осветления. Скройте лишние участки изображения на маске данного корректирующего слоя.

Шаг 110

Продублируйте слой Гребень волны 1 (Spray 1), включая корректирующий слой несколько раз, а затем примените трансформацию к дубликатам слоёв, расположив дубликаты на обоих гребнях волн. Проведите коррекцию на слой-масках. Переименуйте дубликаты слоёв соответственно.

Шаг 111

Повторите ту же самую технику, добавив слой с белой пеной в режиме наложения Осветление (Screen) над головой дельфина. Добавьте ещё один фрагмент пены в области шеи. Поменяйте режим наложения для данного слоя на Перекрытие (Overlay). Проведите коррекцию на слой-масках этих слоёв с пеной. Назовите эти слои Пена 1 (Top foam 1) и Пена 2 (Top foam 2).

Шаг 112

Добавьте новый слой поверх всех остальных слоёв, поменяйте режим наложения для данного слоя на Осветление (Screen). Назовите этот слой кисти Пузырьки (Spray brushes). Выберите инструмент Кисть (Brush Tool (B). Установите Непрозрачность (Opacity) слоя примерно 70%. Далее, с помощью кистей акварель, добавьте пузырьки на оба гребня волн. Чтобы добиться наилучших результатов, поменяйте направление кисти, используйте настройки отражения по оси Х и Y в панели настроек кисти, чтобы избежать повторения.

Шаг 113

Пришло время внести некоторые заключительные изменения. Вначале уменьшите значение Непрозрачности (Opacity) корректирующего слоя Уровни (Levels), который закреплён к слою Основа воды (Base water) до 52%, а затем добавьте корректирующий слой Уровни (Levels) в качестве обтравочной маски к слою Мужчина (Figure).

Шаг 114

Перейдите на слой Небо (Sky), уменьшите Непрозрачность (Opacity) этого слоя до 80%, а затем дважды щёлкните по корректирующему слою Цветовой баланс (Color Balance), который закреплён к этому же слою, чтобы изменить настройки.

Шаг 115

Добавьте новый слой поверх всех остальных слоёв внутри группы Передний план / Задний план (BACKGROUND/FOREGROUND). Назовите этот слой Дымка (Haze). Залейте этот слой бледно-синим цветом (#d2dce9), а затем уменьшите Непрозрачность (Opacity) слоя до 15%. Выберите инструмент Градиент (Gradient (G), установите тип градиента от Основного (чёрного) к Прозрачному,, а также выберите Линейный (Linear) градиент. Далее, перейдите на маску слой Дымка (Haze), чтобы применить градиент в нижней части изображения.

Шаг 116

Для естественного вида теней, добавьте немного шума с помощью фильтра (Noise) на оба слоя с тенью мужчины.

Шаг 117

Создайте новый слой поверх всех слоёв с группами, назовите этот слой Ретушь (Retouch). Теперь с помощью кисти с различной степенью непрозрачности, добавьте бледного цвета пену и фрагменты воды, не забывайте нажимать клавишу (alt) для отбора образцов цветовых оттенков с изображения на своё усмотрение. Для наглядности, я показал этот слой на чёрном фоне на скриншоте ниже.

Также с помощью инструмента Штамп (Clone Stamp Tool) в режиме Активный и ниже (Current & Below), исправьте любые незначительные дефекты на данном слое.

Шаг 118

Далее мы применим выборочное усиление резкости. Нажмите клавиши (Ctrl+Shift+Alt+E), чтобы создать новый объединённый слой поверх всех остальных слоёв, назовите этот слой Усиление резкости (Sharpened), а затем поменяйте режим наложения для данного слоя на Перекрытие (Overlay). Итак, идём Фильтр - Другое - Цветовой контраст (Filter > Other > High Pass). Установите радиус резкости на 2рх.

Шаг 119

Нам нужно, чтобы эффект резкости был применим только элементам переднего плана, поэтому, добавьте слой-маску и с помощью большой мягкой кисти чёрного цвета, прокрасьте поверх заднего плана, чтобы скрыть эффект резкости. Теперь, удерживая клавишу (Ctrl), щёлкните по слою с Голова дельфина (Dolphin head), чтобы загрузить активное выделение, а затем залейте выделение белым цветом на маске. Проделайте то же самое со слоем Мужчина (Figure).

Шаг 120

В заключение, мы проведём общую цветовую коррекцию. Добавьте корректирующий слой Выборочная коррекция цвета (Selective Color) поверх всех остальных слоёв. Не преобразовывайте данный корректирующий слой в обтравочнуцю маску. Установите следующие настройки, которые указаны на скриншоте ниже.

Заключение

Надеюсь, что вам понравился этот урок и вы узнали про новые техники. Теперь, когда вы знаете, как создаётся подобная сцена, почему бы вам не создать свою собственную картину?
